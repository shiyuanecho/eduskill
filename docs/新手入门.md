# edu 新手入门

你可能是看到有人用 edu 推进真实的育儿 / 教学任务，想知道自己该怎么开始。

edu 由拾元创建，是一套覆盖 3-9 岁儿童学、玩、行为、习惯、选课与成长记录的专业工具箱，内置 846 个知识原子与 36 个 Skill。你不需要先读懂全部 36 个 Skill，也不用把问题说得很专业。

先把真实处境交出来，edu 会帮你找到切入点。

## 第一次怎么用

第一次使用，直接输入：

```text
/edu 新手入门
```

edu 会说明你能交付什么、系统怎样处理、可能得到什么结果。你提交真实任务后，它会选择合适的 Skill；每次只决定当前一步，后续方向由实际结果决定。

你也可以输入 `/edu`，然后提交任何真实任务：

```text
我有一个问题还没想清楚：……
这段内容／材料想请你处理：……
我需要判断一个选择：……
这件事一直推进不下去：……
```

例如：

```text
/edu 我家娃 4 岁，数数能数到 20，但让他拿 3 个积木总拿不准，是数感没起来吗？
```

`/edu` 会读当前对话里已经给出的信息，直接把你带到合适的 Skill。你无需重复描述背景。

## 已经知道要做什么

熟悉 edu 后，可以跳过 `/edu`，直接调用具体 Skill。命令后面直接写材料或问题：

```text
/edu-diagnosis 我做中班老师，几个孩子总是不会轮流玩，是社交问题还是规则没建立？
/edu-play 我想让孩子认识形状，有什么能玩的活动？
/edu-talk 陪娃读绘本时，我该怎么提问才能引发他思考？
```

每个 Skill 都会继续追问真正缺少的信息。已经出现在当前对话里的信息，不需要重新回答。

## edu 怎样工作

1. 你提交当前的处境、目标、材料和卡点。
2. `/edu` 判断当前最需要完成的任务，并调用一个 Skill。
3. Skill 直接产出观察、方案、话术、文件或行动。
4. 你补充反馈后，再由 `/edu` 根据新结果判断下一步。

熟悉具体 Skill 后可以直接调用；第一次使用时，直接把任务交给 `/edu` 即可。

## Skill 全目录

以下列出当前正式发布的 36 个 Skill。每项均包含适用时机、可直接输入的话与常见衔接方式。

### 诊断层

#### `/edu-diagnosis`｜学力诊断

**什么时候用：** 孩子在某个具体学习 / 成长任务（数数、分类、比较、空间、语言、社交、习惯、自理等）里卡住了，你想看清卡在哪、属于哪个发展维度。

**你可以这样说：** `/edu-diagnosis 我家娃 4 岁，数数能数到 20，但拿 3 个积木总拿不准，是数感没起来吗？`

**你会得到：** 观察记录（卡点在哪、属于哪个发展维度，是正常阶段还是需关注信号）+ 2-3 个下一阶段的小活动建议。

**常见衔接：** 卡点清楚了用 `/edu-play` 变成活动；日常不知道怎么陪用 `/edu-talk`；涉及行为习惯用 `/edu-behavior`。

#### `/edu-behavior`｜行为情绪诊断

**什么时候用：** 孩子发脾气、打人、黏人、分离焦虑、不肯睡等行为情绪问题，你想看行为背后的原因。

**你可以这样说：** `/edu-behavior 我家娃上幼儿园就黏着不肯进教室，是分离焦虑还是别的问题？`

**你会得到：** 行为背后的可能原因梳理 + 应对方向（不贴标签、不下能力判定）。

**常见衔接：** 行为背后是作息没建立用 `/edu-routine`；需要翻译给家长 / 老师用 `/edu-communicate`；其实是能力卡点表现用 `/edu-diagnosis`。

#### `/edu-content`｜内容诊断

**什么时候用：** 你拿不准一个活动 / 绘本 / 视频是不是真能发展能力，还是假启蒙、纯消遣。

**你可以这样说：** `/edu-content 我想用某识字 App 给 4 岁娃练字，这算真启蒙吗？`

**你会得到：** 对这份内容是否真能发展能力的判断 + 改进或替代方向。

**常见衔接：** 方向定了要变成活动用 `/edu-play`；涉及怎么讲概念用 `/edu-explain`；想盘整体体系用 `/edu-benchmark`。

#### `/edu-anxiety`｜焦虑诊断

**什么时候用：** 你作为家长有一条焦虑，想知道合不合理、值不值得慌。

**你可以这样说：** `/edu-anxiety 周围人都给孩子报思维课，我不报是不是耽误了？`

**你会得到：** 这条焦虑的合理性判断（真风险 vs 抢跑焦虑）+ 该关注的信号。

**常见衔接：** 理清后想知道让孩子获得啥用 `/edu-aim`；涉及选课用 `/edu-decision`；源自体系不确定用 `/edu-benchmark`。

#### `/edu-aim`｜目标理清

**什么时候用：** 你说「想让孩子更专注」「想培养阅读习惯」，却无法确认具体要完成什么。

**你可以这样说：** `/edu-aim 我希望孩子以后有内驱力，现在该做什么？`

**你会得到：** 原目标的语义梳理、可检查的重写版、验收条件和第一步。

**常见衔接：** 目标清楚了要落地用 `/edu-play`；涉及体系覆盖用 `/edu-benchmark`；在几个资源间犹豫用 `/edu-decision`。

#### `/edu-benchmark`｜体系对标

**什么时候用：** 你想盘现在的启蒙 / 教学体系缺什么，或判断自己覆盖得全不全。

**你可以这样说：** `/edu-benchmark 我家 5 岁娃目前主要在认字和算数，体系上还缺哪块？`

**你会得到：** 覆盖度差距地图，指出缺的是运动、社交、习惯还是其他。

**常见衔接：** 差距清楚了要补活动用 `/edu-play`；想记录一次变化用 `/edu-report`。

### 方案层

#### `/edu-play`｜活动 / 游戏设计

**什么时候用：** 你有一个目标（认知、运动、习惯等），想把它拆成能玩的活动。

**你可以这样说：** `/edu-play 我想让 3 岁娃认识颜色，有什么能玩的活动？`

**你会得到：** 2-3 个具体活动（玩法、材料、怎么观察孩子反应）。

**常见衔接：** 过程中发现新卡点回 `/edu-diagnosis`；想记一笔用 `/edu-report`；想存长期档案用 `/edu-archive`。

#### `/edu-talk`｜陪玩话术

**什么时候用：** 陪玩 / 生活中你不知道怎么问、怎么接，想有现成话术。

**你可以这样说：** `/edu-talk 陪娃搭积木时，我该怎么提问才能引发他思考？`

**你会得到：** 一组现成的提问与回应话术（分家长版 / 老师版）。

**常见衔接：** 话术涉及讲概念用 `/edu-explain`；想记一笔用 `/edu-report`。

#### `/edu-explain`｜概念讲法

**什么时候用：** 一个抽象概念（如「公平」「时间」「死亡」）你不知道怎么讲给孩子听。

**你可以这样说：** `/edu-explain 怎么给 5 岁孩子讲「为什么不能一直看电视」？`

**你会得到：** 把抽象概念讲成孩子能懂的具体讲法 + 示例。

**常见衔接：** 讲法要落地成活动用 `/edu-play`；话术配合用 `/edu-talk`。

#### `/edu-question`｜提问设计

**什么时候用：** 你想围绕一个主题，设计一组能引发孩子思考的好问题。

**你可以这样说：** `/edu-question 围绕「春天」给幼儿园孩子设计几个好问题。`

**你会得到：** 一组分层问题（从观察到推理到表达）。

**常见衔接：** 问题配合活动用 `/edu-play`；话术配合用 `/edu-talk`。

#### `/edu-routine`｜一日流程与常规

**什么时候用：** 你想建立或调整作息、过渡环节、常规（如收玩具、饭前洗手）。

**你可以这样说：** `/edu-routine 我家娃晚上不肯睡，怎么建立睡前常规？`

**你会得到：** 可落地的流程与常规建立步骤 + 过渡环节话术。

**常见衔接：** 常规建立后想记录用 `/edu-report`；沟通发现底层能力问题用 `/edu-diagnosis`。

#### `/edu-communicate`｜家园沟通

**什么时候用：** 你（家长或老师）需要把对孩子的观察翻译给另一方，听得懂、不对立。

**你可以这样说：** `/edu-communicate 怎么跟家长说孩子上课总走神，又不显得在告状？`

**你会得到：** 分家长版 / 老师版的沟通话术与结构。

**常见衔接：** 沟通发现底层能力问题回 `/edu-diagnosis`；想记录用 `/edu-report`。

### 决策层

#### `/edu-decision`｜选课决策

**什么时候用：** 你在几个课程 / 资源之间犹豫，想做不后悔的选择。

**你可以这样说：** `/edu-decision 思维课 A 和 B 价格差不多，怎么选？`

**你会得到：** 判断框架 + 针对你情况的取舍建议。

**常见衔接：** 决策后想记录理由用 `/edu-report`；决策涉及目标重审用 `/edu-aim`。

### 记录层

#### `/edu-report`｜进展记录

**什么时候用：** 当前对话已经得出关键结论，你想记一笔，下次回看。

**你可以这样说：** `/edu-report` 或 `/edu-report 乐乐数感进展`

**你会得到：** 一份结构化观察记录（结论、做了什么、下一步）。

**常见衔接：** 想存进长期档案用 `/edu-archive`；发现未解问题回对应诊断 Skill。

#### `/edu-archive`｜成长档案

**什么时候用：** 你想给某个孩子建一份能长期回看的档案，跨会话追加。

**你可以这样说：** `/edu-archive 给乐乐建一份成长档案`

**你会得到：** 当前工作目录下的 `edu-档案/<孩子代号>.md`，每次更新为追加。

**常见衔接：** 档案里发现未解问题回对应诊断 / 方案 Skill。

### 教师实务层

#### `/edu-lesson`｜备课 / 教案设计

**什么时候用：** 你要上一节课或开展一个活动，需要把「想让孩子获得什么 / 上什么内容」落成一份能直接上的教案。

**你可以这样说：** `/edu-lesson 下周要给中班做一节「按数量配对」的数学活动，帮我写教案`

**你会得到：** 含学情分析、分层目标、活动环节（导入—展开—收尾）、材料准备、弹性预案、家长在家延伸版的完整教案。

**常见衔接：** 教案要变成活动细节用 `/edu-play`；要观察孩子反应用 `/edu-observe`；班级差异大用 `/edu-differentiate`。

#### `/edu-comment`｜写评语

**什么时候用：** 期末 / 阶段末要写给孩子或家长的评语、学习故事、家园联系册一句话，想写得出差异化、不套话。

**你可以这样说：** `/edu-comment 给一个语言能力很强但规则意识弱的中班男孩写学期评语`

**你会得到：** 观察记录式 / 学习故事式 / 一句话式评语模板，按孩子特点差异化生成，不贴标签。

**常见衔接：** 评语依据来自 `/edu-observe`；要同步给家长用 `/edu-notice`。

#### `/edu-observe`｜观察评估模板库

**什么时候用：** 你需要做规范的儿童观察记录（叙事 / 时间取样 / 事件取样 / 检核表 / 量表），或要一张发展检核清单、学习故事模板。

**你可以这样说：** `/edu-observe 给我一张 4-5 岁社会性发展检核表`

**你会得到：** 可直接抄用的观察记录方法与模板、发展检核清单、学习故事模板、作品分析表。

**常见衔接：** 观察结论要判定卡点回 `/edu-diagnosis`；要写成评语用 `/edu-comment`；要存档案用 `/edu-archive`。

#### `/edu-differentiate`｜差异化与个别引导

**什么时候用：** 一个班孩子差异大，有人「吃不饱」有人「跟不上」，或你有攻击 / 退缩 / 多动 / 规则弱四类典型儿童要针对性引导。

**你可以这样说：** `/edu-differentiate 班里有个总抢玩具的男孩，怎么分层引导？`

**你会得到：** 分层分组方案、学习风格 / 气质适配思路、IEP 雏形，以及四类儿童的正确引导模板。

**常见衔接：** 引导要落到活动用 `/edu-play`；涉及特殊需要转 `/edu-inclusive`。

#### `/edu-inclusive`｜特殊需要融合支持

**什么时候用：** 班里有随班就读或疑似特殊需要的孩子，你需要融合教育政策依据、IEP/ISP 思路、常见障碍支持策略与转介流程。

**你可以这样说：** `/edu-inclusive 班里有个语迟的孩子，随班就读我该怎么支持？`

**你会得到：** 融合政策依据、常见障碍（孤独症 / ADHD / 语迟 / 感统）支持策略清单、资源教室与转介流程模板。

**常见衔接：** 筛查非诊断，疑点转介专业；家庭侧配合用 `/edu-specialhome`。

#### `/edu-safety`｜课堂突发与安全应急

**什么时候用：** 遇到同伴冲突、情绪崩溃、离座乱跑、安全先兆或生理急症（呕吐 / 过敏 / 惊厥），需要马上能照着做的步骤。

**你可以这样说：** `/edu-safety 一个孩子突然抽搐，我第一时间该做什么？`

**你会得到：** 分场景应急步骤卡、对家长的话术、上报流程（含医疗免责提示，不替代专业急救）。

**常见衔接：** 事后要跟家长沟通用 `/edu-notice`；要记一笔用 `/edu-report`。

#### `/edu-reflect`｜教研反思与写作

**什么时候用：** 你要做活动复盘、写园本教研发言 / 记录，或写教研文章 / 案例，想要能直接抄的框架。

**你可以这样说：** `/edu-reflect 帮我写这次建构区活动的教研复盘`

**你会得到：** 活动复盘模板（目标—过程—幼儿反应—调整）、园本教研发言 / 记录模板、教研文章 / 案例写作框架。

**常见衔接：** 复盘发现孩子卡点回 `/edu-diagnosis`；要落教案用 `/edu-lesson`。

#### `/edu-material`｜教具学具与课件

**什么时候用：** 你要准备低结构材料玩法、自制教具清单，或做课件 PPT、找数字化资源，想重实物操作、课件只是辅助。

**你可以这样说：** `/edu-material 给我一份「分类」主题的低结构材料玩法清单`

**你会得到：** 低结构材料玩法、自制教具清单、课件 PPT 制作原则、数字化资源生成思路。

**常见衔接：** 材料要变成活动用 `/edu-play`；环境布置用 `/edu-env`。

#### `/edu-env`｜环境创设

**什么时候用：** 你要做区角划分、材料投放与标签、主题墙「会说话」、过渡环节设计，让环境成为「第三位老师」。

**你可以这样说：** `/edu-env 帮我规划中班教室的区角划分`

**你会得到：** 区角划分、材料投放与标签、主题墙、过渡环节的做法与清单模板。

**常见衔接：** 区角活动设计用 `/edu-play`；材料准备用 `/edu-material`。

#### `/edu-notice`｜家园共育与家长会

**什么时候用：** 你要跟家长做非暴力沟通、给正向具体反馈，或要办家长会、写给家长的一封信。

**你可以这样说：** `/edu-notice 帮我写这周五家长会的发言稿`

**你会得到：** 非暴力沟通话术、正向具体反馈结构、各场景话术对照、家长会流程与发言稿、给家长的一封信模板。

**常见衔接：** 反馈依据来自 `/edu-observe`；要讲家庭指导用 `/edu-workshop`。

#### `/edu-workshop`｜家长课堂 / 讲座素材

**什么时候用：** 你要给家长做讲座 / 线上分享，需要主题框架、PPT 大纲、讲稿要点、互动环节和一页纸干货。

**你可以这样说：** `/edu-workshop 给我一场「如何陪孩子读绘本」家长讲座的大纲`

**你会得到：** 主题框架、PPT 大纲、讲稿要点、互动环节、给家长的一页纸。

**常见衔接：** 讲座要落到沟通用 `/edu-parenttalk`；要配套信件用 `/edu-notice`。

### 家长实务层

#### `/edu-studyplan`｜学习与阅读规划

**什么时候用：** 你要给孩子规划兴趣班取舍、分级阅读与桥梁书过渡、在家学习节奏，想要能直接套的清单。

**你可以这样说：** `/edu-studyplan 我家娃 5 岁，英语和思维课怎么取舍？`

**你会得到：** 兴趣班取舍清单、分级阅读与桥梁书过渡方案、在家学习节奏模板（仅给规划，不下能力判定）。

**常见衔接：** 阅读要落活动用 `/edu-play`；目标不清用 `/edu-aim`。

#### `/edu-screen`｜作息与屏幕规划

**什么时候用：** 你要排一日作息表、周末假期安排、定家庭屏幕公约，或做幼小衔接四大准备。

**你可以这样说：** `/edu-screen 给我一份 4 岁娃的周末作息和屏幕公约`

**你会得到：** 一日作息表、周末假期安排、家庭屏幕公约、幼小衔接准备模板（时长建议以官方最新为准）。

**常见衔接：** 作息要落常规用 `/edu-routine`；衔接专项用 `/edu-transition`。

#### `/edu-time`｜家庭时间管理与任务执行

**什么时候用：** 孩子磨蹭、任务执行差，你要视觉计时器、任务清单、习惯绑定、正强化等能直接用的工具。

**你可以这样说：** `/edu-time 娃写作业一直磨蹭，怎么给他外挂时间支架？`

**你会得到：** 视觉计时器、任务清单、习惯绑定、正强化等可直接用的模板（外挂支架，不靠硬管）。

**常见衔接：** 任务涉及常规用 `/edu-routine`；习惯培养用 `/edu-rules`。

#### `/edu-parenttalk`｜亲子沟通与情绪引导

**什么时候用：** 日常冲突下你不知道怎么说孩子才接得住，或要引导情绪、处理闹着不走 / 两娃抢玩具 / 不肯睡。

**你可以这样说：** `/edu-parenttalk 娃在大街上闹着不走，我该怎么说？`

**你会得到：** 「我信息」「启发式提问」话术卡、Gottman 情绪教练五步、典型场景应对。

**常见衔接：** 沟通背后是习惯用 `/edu-rules`；是作息用 `/edu-routine`。

#### `/edu-rules`｜家庭规则与陪伴质量

**什么时候用：** 你想定「和善而坚定」的少量核心规则，或用专属时间 / 地板时间提升陪伴质量、养用餐睡眠如厕习惯。

**你可以这样说：** `/edu-rules 帮我定 3 条家里最核心的规则`

**你会得到：** 规则制定模板 + 陪伴活动清单 + 自理能力里程碑表。

**常见衔接：** 规则要落地执行用 `/edu-time`；沟通配合用 `/edu-parenttalk`。

#### `/edu-schoolchoice`｜选园选校

**什么时候用：** 你要考察幼儿园 / 小学、自查合理期望（ZPD / 成长型思维）、调解手足冲突、识别发展预警（非诊断）。

**你可以这样说：** `/edu-schoolchoice 帮我校对一所民办幼儿园的考察清单`

**你会得到：** 园 / 校考察清单、合理期望自查、家长情绪暂停、发展预警识别（非诊断）。

**常见衔接：** 预警疑点转 `/edu-specialhome`；衔接用 `/edu-transition`。

#### `/edu-transition`｜幼小衔接专项

**什么时候用：** 孩子快上小学，你要按身心 / 生活 / 社会 / 学习四大准备做科学衔接，反对提前刷题抢跑。

**你可以这样说：** `/edu-transition 大班娃幼升小，家庭衔接计划怎么定？`

**你会得到：** 四大准备衔接计划、衔接质量观察清单（看行为不考试）。

**常见衔接：** 准备涉及作息用 `/edu-screen`；涉及习惯用 `/edu-rules`。

#### `/edu-ages`｜分龄发展养育指南

**什么时候用：** 你想按 3-4 / 4-5 / 5-6 / 6-9 各龄看里程碑与养育重点，或判断「孩子是不是落后」。

**你可以这样说：** `/edu-ages 4 岁孩子应该会什么？`

**你会得到：** 分龄速查表（身体 / 语言 / 认知 / 社会性）、危险信号清单（非诊断，疑点转介专业）。

**常见衔接：** 疑点转 `/edu-specialhome`；要定计划用 `/edu-screen` 或 `/edu-rules`。

#### `/edu-specialhome`｜特殊需要儿童家庭指导

**什么时候用：** 你怀疑或已知孩子有发育 / 特殊需要，要识别预警、找早期干预路径、做家庭日常支持、配合机构学校。

**你可以这样说：** `/edu-specialhome 娃 2 岁半还不太说话，家里能做什么？`

**你会得到：** 发育预警识别、早期干预路径与依据、家庭支持策略清单、与机构 / 学校配合要点（筛查非诊断，转介专业）。

**常见衔接：** 预警来源可能是 `/edu-ages`；学校侧配合用 `/edu-inclusive`。

#### `/edu-coparent`｜父亲参与与隔代教养

**什么时候用：** 你想提升父亲参与、理清隔代教养边界、定三代协作规则、化解教养冲突。

**你可以这样说：** `/edu-coparent 爷爷奶奶总宠娃，怎么定三代协作规则？`

**你会得到：** 父亲可参与事务清单、隔代教养特点与边界、三代协作规则、教养冲突化解模板。

**常见衔接：** 冲突涉及规则用 `/edu-rules`；涉及沟通用 `/edu-parenttalk`。

## 常见场景与动态导航

edu 不预设固定执行链。每次只调用当前最合适的一个 Skill；完成后输入 `/edu`，它会结合刚才的结论、你的补充和目标变化，重新判断下一步。

### 1. 从零开始做启蒙

```text
/edu
```

先说清孩子年龄、你关注什么。当前问题可能进入学力诊断、活动设计、概念讲法或常规建立；由当前结论决定。

### 2. 判断一个孩子卡在哪

```text
/edu-diagnosis
```

先拆清任务和孩子表现。诊断后的结果可能需要设计活动、练习陪玩话术，或回到行为情绪看根子；由诊断结论决定。

### 3. 把目标变成能玩的活动

```text
/edu-play
```

先说清你想让孩子获得什么。活动设计完后，下一步可能是记一笔或存进档案。

### 4. 家长的一条焦虑

```text
/edu-anxiety
```

先判断这条焦虑合不合理。结论会指出当前更优先理清目标、做选课决策，还是盘体系。

### 5. 建立作息与常规

```text
/edu-routine
```

先看清当前作息断点。常规建立后，下一步可能是记录或沟通。

### 6. 给一个孩子建成长档案

```text
/edu-archive
```

先把观察与结论存成文件。后续行动根据档案内容和当前任务重新判断。

## 使用习惯

- **材料越真实，结果越具体。** 说清楚孩子多大、面对啥、你已经试过啥、哪里失败了。
- **不用重复背景。** `/edu` 和各个 Skill 应先利用当前对话里的信息；只补充真正缺少的事实。
- **不知道下一步时回到 `/edu`。** 它会按当前结论推荐值得继续走的方向。
- **一次只决定一步。** 当前结论和你的新反馈会改变下一次路由，因而不预先安排固定长链。

## 护栏（所有 Skill 都守）

- 只做观察记录，不下儿童能力判定结论。
- 凡涉及儿童发展的结论，尾部带非定论标识：「观察记录，非测评结论；重大安排请咨询早教 / 幼教专业人士」。
- 知识引权威（发展指南、课标、发展心理学共识），不编造来源。
- 说人话，不加表情符号。
