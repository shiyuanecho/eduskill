# edu 新手入门

你可能是看到有人用 edu 推进真实的育儿 / 教学任务，想知道自己该怎么开始。

edu 由拾元创建，是一套覆盖 0-9 岁儿童学、玩、行为、习惯、选课与成长记录的专业工具箱，内置 846 个知识原子与 23 个 Skill。你不需要先读懂全部 23 个 Skill，也不用把问题说得很专业。

先把真实处境交出来，edu 会帮你找到切入点。

## 第一次怎么用

第一次使用，直接输入：

```text
/edu 新手入门
```

edu 会说明你能交付什么、系统怎样处理、可能得到什么结果。你提交真实任务后，它会选择合适的 Skill；每次只决定当前一步，后续方向由实际结果决定。

你也可以输入 `/edu`，然后提交任何真实任务：

```text
我有一个问题还没想清楚：……
这段内容／材料想请你处理：……
我需要判断一个选择：……
这件事一直推进不下去：……
```

例如：

```text
/edu 我家娃 4 岁，数数能数到 20，但让他拿 3 个积木总拿不准，是数感没起来吗？
```

`/edu` 会读当前对话里已经给出的信息，直接把你带到合适的 Skill。你无需重复描述背景。

## 已经知道要做什么

熟悉 edu 后，可以跳过 `/edu`，直接调用具体 Skill。命令后面直接写材料或问题：

```text
/edu-diagnosis 我做中班老师，几个孩子总是不会轮流玩，是社交问题还是规则没建立？
/edu-lesson 我想让 3 岁娃认识颜色，怎么设计一个启蒙活动？
/edu-talk 陪娃读绘本时，我该怎么提问才能引发他思考？
```

每个 Skill 都会继续追问真正缺少的信息。已经出现在当前对话里的信息，不需要重新回答。

## edu 怎样工作

1. 你提交当前的处境、目标、材料和卡点。
2. `/edu` 判断当前最需要完成的任务，并调用一个 Skill。
3. Skill 直接产出观察、方案、话术、文件或行动。
4. 你补充反馈后，再由 `/edu` 根据新结果判断下一步。

熟悉具体 Skill 后可以直接调用；第一次使用时，直接把任务交给 `/edu` 即可。

## Skill 全目录

当前正式发布 23 个 Skill，按任务类型分成 5 层。每项均包含适用时机、可直接输入的话与常见衔接方式。

### 诊断层（9 个）

#### `/edu-diagnosis`｜学力诊断

**什么时候用：** 孩子在某个具体学习 / 成长任务（数数、分类、比较、空间、语言、社交、习惯、自理等）里卡住了，你想看清卡在哪、属于哪个发展维度。

**你可以这样说：** `/edu-diagnosis 我家娃 4 岁，数数能数到 20，但拿 3 个积木总拿不准，是数感没起来吗？`

**你会得到：** 观察记录（卡点在哪、属于哪个发展维度，是正常阶段还是需关注信号）+ 2-3 个下一阶段的小活动建议。

**常见衔接：** 卡点清楚了用 `/edu-lesson` 变成活动；日常不知道怎么陪用 `/edu-talk`；涉及行为习惯用 `/edu-behavior`。

#### `/edu-behavior`｜行为情绪诊断

**什么时候用：** 孩子发脾气、打人、黏人、分离焦虑、不肯睡等行为情绪问题，你想看行为背后的原因。

**你可以这样说：** `/edu-behavior 我家娃上幼儿园就黏着不肯进教室，是分离焦虑还是别的问题？`

**你会得到：** 行为背后的可能原因梳理 + 应对方向（不贴标签、不下能力判定）。

**常见衔接：** 行为背后是家庭节奏没建立用 `/edu-family`；需要翻译给家长 / 老师用 `/edu-connect`；其实是能力卡点表现用 `/edu-diagnosis`。

#### `/edu-content`｜内容诊断

**什么时候用：** 你拿不准一个活动 / 绘本 / 视频是不是真能发展能力，还是假启蒙、纯消遣。

**你可以这样说：** `/edu-content 我想用某识字 App 给 4 岁娃练字，这算真启蒙吗？`

**你会得到：** 对这份内容是否真能发展能力的判断 + 改进或替代方向。

**常见衔接：** 方向定了要变成活动用 `/edu-lesson`；想盘整体体系用 `/edu-benchmark`。

#### `/edu-anxiety`｜焦虑诊断

**什么时候用：** 你作为家长有一条焦虑，想知道合不合理、值不值得慌。

**你可以这样说：** `/edu-anxiety 周围人都给孩子报思维课，我不报是不是耽误了？`

**你会得到：** 这条焦虑的合理性判断（真风险 vs 抢跑焦虑）+ 该关注的信号。

**常见衔接：** 理清后想知道让孩子获得啥用 `/edu-aim`；涉及选课用 `/edu-decision`；源自体系不确定用 `/edu-benchmark`。

#### `/edu-aim`｜目标理清

**什么时候用：** 你说「想让孩子更专注」「想培养阅读习惯」，却无法确认具体要完成什么。

**你可以这样说：** `/edu-aim 我希望孩子以后有内驱力，现在该做什么？`

**你会得到：** 原目标的语义梳理、可检查的重写版、验收条件和第一步。

**常见衔接：** 目标清楚了要落地用 `/edu-lesson`；涉及体系覆盖用 `/edu-benchmark`；在几个资源间犹豫用 `/edu-decision`。

#### `/edu-benchmark`｜体系对标

**什么时候用：** 你想盘现在的启蒙 / 教学体系缺什么，或判断自己覆盖得全不全。

**你可以这样说：** `/edu-benchmark 我家 5 岁娃目前主要在认字和算数，体系上还缺哪块？`

**你会得到：** 覆盖度差距地图，指出缺的是运动、社交、习惯还是其他。

**常见衔接：** 差距清楚了要补活动用 `/edu-lesson`；想记录一次变化用 `/edu-report`。

#### `/edu-observe`｜观察评估模板库

**什么时候用：** 你需要做规范的儿童观察记录（叙事 / 时间取样 / 事件取样 / 检核表 / 量表），或要一张发展检核清单、学习故事模板。

**你可以这样说：** `/edu-observe 给我一张 4-5 岁社会性发展检核表`

**你会得到：** 可直接抄用的观察记录方法与模板、发展检核清单、学习故事模板、作品分析表。

**常见衔接：** 观察结论要判定卡点回 `/edu-diagnosis`；要写成评语用 `/edu-comment`；要存档案用 `/edu-archive`。

#### `/edu-ages`｜分龄发展养育指南

**什么时候用：** 你想按 3-4 / 4-5 / 5-6 / 6-9 各龄看里程碑与养育重点，或判断「孩子是不是落后」。

**你可以这样说：** `/edu-ages 4 岁孩子应该会什么？`

**你会得到：** 分龄速查表（身体 / 语言 / 认知 / 社会性）、危险信号清单（非诊断，疑点转介专业）。

**常见衔接：** 疑点转 `/edu-inclusive` 看融合支持；要定计划用 `/edu-family`。

#### `/edu-grow`｜发展科学与脑科学

**什么时候用：** 你想解释孩子行为背后的科学原理，了解皮亚杰、维果茨基、执行功能、成长型思维等关键概念。

**你可以这样说：** `/edu-grow 为什么孩子总爱反复问同一个问题？`

**你会得到：** 把发展科学 / 脑科学 / 学习科学原理讲成能用的方法，带实际做法。

**常见衔接：** 原理想落到活动用 `/edu-lesson`；想结合分龄里程碑看用 `/edu-ages`。

### 方案层（10 个）

#### `/edu-lesson`｜教案与活动设计

**什么时候用：** 你有一个目标（认知、运动、习惯等），想把它拆成能玩的教案或活动；也覆盖概念讲法、提问设计、教具、环境布置。

**你可以这样说：** `/edu-lesson 我想让 3 岁娃认识颜色，怎么设计一个启蒙活动？`

**你会得到：** 2-3 个具体活动或一份完整教案（玩法、材料、怎么观察孩子反应、家庭延伸版）。

**常见衔接：** 过程中发现新卡点回 `/edu-diagnosis`；想记一笔用 `/edu-report`；想存长期档案用 `/edu-archive`。

#### `/edu-talk`｜亲子沟通与情绪引导

**什么时候用：** 陪玩 / 生活中你不知道怎么问、怎么接，想有现成话术；或处理孩子情绪、冲突场景。

**你可以这样说：** `/edu-talk 陪娃搭积木时，我该怎么提问才能引发他思考？`

**你会得到：** 一组现成的提问与回应话术（分家长版 / 老师版），含情绪引导场景。

**常见衔接：** 话术涉及讲概念用 `/edu-lesson`；想记一笔用 `/edu-report`。

#### `/edu-classroom`｜班级与课堂管理

**什么时候用：** 你要处理班级日常秩序、安全应急、常规建立、差异化与分层引导。

**你可以这样说：** `/edu-classroom 孩子起冲突了，我该怎么分层处理？`

**你会得到：** 班级管理策略、常规建立步骤、安全应急流程、分层引导方案。

**常见衔接：** 发现孩子能力卡点回 `/edu-diagnosis`；需要特殊需要支持用 `/edu-inclusive`；要记一笔用 `/edu-report`。

#### `/edu-family`｜家庭日常管理

**什么时候用：** 你要安排作息 / 屏幕时间 / 假期规划 / 时间管理 / 任务执行 / 家庭规则 / 陪伴质量 / 生活习惯 / 学习与阅读规划。

**你可以这样说：** `/edu-family 给我一份 4 岁娃的周末作息和屏幕公约`

**你会得到：** 可落地的家庭节奏、规则、时间支架与陪伴方案。

**常见衔接：** 规则执行不下去看 `/edu-behavior`；想记录变化用 `/edu-report`；幼小衔接专项用 `/edu-connect`。

#### `/edu-connect`｜家园协作与衔接

**什么时候用：** 你要选园选校、做幼小衔接、处理父亲参与 / 隔代教养、家园沟通与家长会、给家长的一封信。

**你可以这样说：** `/edu-connect 怎么跟家长说孩子上课总走神，又不显得在告状？`

**你会得到：** 分家长版 / 老师版的沟通话术、选园清单、衔接计划、家长会素材。

**常见衔接：** 沟通发现底层能力问题回 `/edu-diagnosis`；想记录用 `/edu-report`。

#### `/edu-comment`｜写评语

**什么时候用：** 期末 / 阶段末要写给孩子或家长的评语、学习故事、家园联系册一句话，想写得出差异化、不套话。

**你可以这样说：** `/edu-comment 给一个语言能力很强但规则意识弱的中班男孩写学期评语`

**你会得到：** 观察记录式 / 学习故事式 / 一句话式评语模板，按孩子特点差异化生成，不贴标签。

**常见衔接：** 评语依据来自 `/edu-observe`；要同步给家长用 `/edu-connect`。

#### `/edu-teachdev`｜教师专业成长

**什么时候用：** 你要做教研活动策划、教学反思与写作、教师培训、家长课堂 / 讲座素材、专业标准对齐。

**你可以这样说：** `/edu-teachdev 帮我写这次建构区活动的教研复盘`

**你会得到：** 活动复盘模板、教研发言 / 记录模板、培训 / 讲座大纲与素材。

**常见衔接：** 复盘发现孩子卡点回 `/edu-diagnosis`；要落教案用 `/edu-lesson`。

#### `/edu-tutor`｜家庭陪学方法

**什么时候用：** 你要在家教孩子自然拼读、数学、书写、认字等具体学科内容，需要"具体怎么教"的方法。

**你可以这样说：** `/edu-tutor 怎么教 5 岁孩子写数字？`

**你会得到：** 分龄陪学方法、常见错误纠正、教具与游戏化建议。

**常见衔接：** 具体讲法可再拆成活动用 `/edu-lesson`；孩子卡住了回 `/edu-diagnosis`。

#### `/edu-inclusive`｜特殊需要融合支持

**什么时候用：** 班里有随班就读或疑似特殊需要的孩子，或家庭需要发育预警识别、早期干预路径、家庭支持策略。

**你可以这样说：** `/edu-inclusive 班里有个语迟的孩子，随班就读我该怎么支持？`

**你会得到：** 融合政策依据、常见障碍（孤独症 / ADHD / 语迟 / 感统）支持策略清单、资源教室与转介流程模板、家庭配合要点。

**常见衔接：** 筛查非诊断，疑点转介专业；学校侧配合用 `/edu-classroom`；家庭侧用 `/edu-family`。

#### `/edu-infant`｜0-3 岁家庭养育实务

**什么时候用：** 你要处理新生儿护理、睡眠节律、喂养辅食、大运动、口腔感官、如厕训练、发展预警等 0-3 岁养育问题。

**你可以这样说：** `/edu-infant 8 个月宝宝睡眠倒退怎么办？`

**你会得到：** 0-3 岁发展规律、日常养育方案、发展预警速查（非诊断，疑点转介专业）。

**常见衔接：** 预警疑点转 `/edu-inclusive`；作息与家庭节奏用 `/edu-family`。

### 决策层（1 个）

#### `/edu-decision`｜选课决策

**什么时候用：** 你在几个课程 / 资源之间犹豫，想做不后悔的选择。

**你可以这样说：** `/edu-decision 思维课 A 和 B 价格差不多，怎么选？`

**你会得到：** 判断框架 + 针对你情况的取舍建议。

**常见衔接：** 决策后想记录理由用 `/edu-report`；决策涉及目标重审用 `/edu-aim`。

### 记录层（2 个）

#### `/edu-report`｜进展记录

**什么时候用：** 当前对话已经得出关键结论，你想记一笔，下次回看。

**你可以这样说：** `/edu-report` 或 `/edu-report 乐乐数感进展`

**你会得到：** 一份结构化观察记录（结论、做了什么、下一步）。

**常见衔接：** 想存进长期档案用 `/edu-archive`；发现未解问题回对应诊断 / 方案 Skill。

#### `/edu-archive`｜成长档案

**什么时候用：** 你想给某个孩子建一份能长期回看的档案，跨会话追加。

**你可以这样说：** `/edu-archive 给乐乐建一份成长档案`

**你会得到：** 当前工作目录下的 `edu-档案/<孩子代号>.md`，每次更新为追加。

**常见衔接：** 档案里发现未解问题回对应诊断 / 方案 Skill。

### 工具层（1 个）

#### `/edu-bridge`｜多端 Skill 桥接

**什么时候用：** 你想把 edu 这套 Skill 同时挂到 Claude Code、Codex、豆包 Mac App / Trae Solo、Grok 等多个 Agent 端使用。

**你可以这样说：** `/edu-bridge 桥接整个 skills 目录`

**你会得到：** 在目标 Agent 目录创建软链 / bridge 文件，让各端都能调用同一套 Skill 源。

**常见衔接：** 桥接前确保 skills 目录已更新；桥接后可用各端调用 `/edu` 或其他子 Skill。

## 常见场景与动态导航

edu 不预设固定执行链。每次只调用当前最合适的一个 Skill；完成后输入 `/edu`，它会结合刚才的结论、你的补充和目标变化，重新判断下一步。

### 1. 从零开始做启蒙

```text
/edu
```

先说清孩子年龄、你关注什么。当前问题可能进入学力诊断、活动设计、家庭管理或概念讲法；由当前结论决定。

### 2. 判断一个孩子卡在哪

```text
/edu-diagnosis
```

先拆清任务和孩子表现。诊断后的结果可能需要设计活动、练习陪玩话术，或回到行为情绪看根子；由诊断结论决定。

### 3. 把目标变成能玩的活动

```text
/edu-lesson
```

先说清你想让孩子获得什么。活动设计完后，下一步可能是记一笔或存进档案。

### 4. 家长的一条焦虑

```text
/edu-anxiety
```

先判断这条焦虑合不合理。结论会指出当前更优先理清目标、做选课决策，还是盘体系。

### 5. 建立作息与常规

```text
/edu-family
```

先看清当前作息断点、家庭节奏和规则。常规建立后，下一步可能是记录或沟通。

### 6. 给一个孩子建成长档案

```text
/edu-archive
```

先把观察与结论存成文件。后续行动根据档案内容和当前任务重新判断。

## 使用习惯

- **材料越真实，结果越具体。** 说清楚孩子多大、面对啥、你已经试过啥、哪里失败了。
- **不用重复背景。** `/edu` 和各个 Skill 应先利用当前对话里的信息；只补充真正缺少的事实。
- **不知道下一步时回到 `/edu`。** 它会按当前结论推荐值得继续走的方向。
- **一次只决定一步。** 当前结论和你的新反馈会改变下一次路由，因而不预先安排固定长链。

## 护栏（所有 Skill 都守）

- 只做观察记录，不下儿童能力判定结论。
- 凡涉及儿童发展的结论，尾部带非定论标识：「观察记录，非测评结论；重大安排请咨询早教 / 幼教专业人士」。
- 知识引权威（发展指南、课标、发展心理学共识），不编造来源。
- 说人话，不加表情符号。
