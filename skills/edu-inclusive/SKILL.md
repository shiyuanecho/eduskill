# /edu-inclusive 融合教育与特殊需要支持

教师怎么在普通班级支持特殊需要儿童，家长怎么在家识别和支持发育落后或有障碍的孩子。面向教师和家长。

执行前先读：knowledge/skill-packages/24-教师实务Tec.md（第七章「特殊需要融合支持」）和 knowledge/skill-packages/25-家长实务Tec.md（第八章「特殊需要儿童家庭指导」）。

## 做法

1. 根据用户身份判断侧重点——教师侧重"融合实施/IEP/班级策略"，家长侧重"识别/转介/在家支持"
2. **教师侧**：
   - ISP/IEP 简易流程：收集信息→先写"能做什么"→定 3-5 条可观察目标→设计支持策略→分工执行→每 1-3 月复盘
   - 常见障碍支持策略：孤独症谱系（稳定日程+视觉提示+同伴小老师）、ADHD（座位近教师+长任务拆短+即时正向反馈）、语言发育迟缓（扩展法+制造沟通机会+配图沟通板）、感统失调（自我调节角+避免强制触碰）
   - 转介流程：日常观察记录（1-2周）→园内沟通→非正式家园沟通（先优后忧）→基础调整观察 2-4 周→持续存在则建议专业评估
3. **家长侧**：
   - 发育预警识别：对照 CDC 里程碑，定期观察记录，相信自己的直觉——"总觉得不对"就记录并咨询
   - 早期干预路径：儿科/发育行为儿科评估→拿评估报告→申请康复救助→机构干预+家庭日常泛化
   - 在家支持：面对面平视慢语速、从兴趣出发、扩充法（孩子说"车"家长补"红色小车"）、短时高频（5-15分钟/次）、正向行为支持
   - 家庭心理：给手足平等关注（"弟弟大脑学习方式不同，需要更多练习"）、家长自建支持网络、"我们也在学习"
4. **通用原则**：调整的是"方式"不是"降低标准"；优先用最少量的辅助让孩子参与→逐步撤除

## 输出格式

- ISP 模板：优势→困难→3-5条可观察目标→支持策略→负责人→复盘日期
- 策略卡：障碍类型→支持策略（环境/教学/行为/同伴）→一句话给配班/保育/家长
- 转介路径：观察→园内沟通→家园沟通→班级调整→专业评估→更新方案→持续随访
- 家庭咨询：观察记录表模板→就医准备清单→家校沟通要点

## 护栏

- 教师无权做医学诊断：任何"你家孩子是不是XX障碍"改为"我们在某方面观察到一些需要多关注的表现，建议去专业机构评估确认"
- 发育预警征≠确诊，只代表"值得尽快评估"
- 不制造恐慌："建议带孩子去做一次发育筛查"≠"你的孩子有问题"
- 去污名化：用"需要更多支持/擅长的不一样"替代"有病/笨/不正常"
