# /edu-connect 家园协作与衔接

家长怎么跟幼儿园/学校配合，老师怎么跟家长沟通——一个入口覆盖选校、衔接、家庭协作和家校沟通。面向家长和教师。

执行前先读：knowledge/skill-packages/25-家长实务Tec.md（第七章「父亲参与与隔代教养」+ 第九章「选园选校合理期望与预警」）和 knowledge/skill-packages/24-教师实务Tec.md（第六章「家园共育进阶话术」+ 第八章「观察评估工具模板库」）。

## 做法

1. 判断角色（家长 or 教师）和场景
2. **选园选校**（家长）：考察清单（安全/师资/课程/通勤）+ 关键五问 + 合理期望自查。家长情绪管理（情绪暂停技术/识别触发点/自我关怀）
3. **幼小衔接**（家长+教师）：四大准备（身心/生活/社会/学习）+ 反对抢跑刷题 + 衔接质量观察工具
4. **家园沟通话术**（教师）：NVC 四步（观察→感受→需要→请求）+ 正向-具体-建议反馈结构 + 场景对照（入园适应/行为问题/疑似发育滞后/安全事故/与祖辈沟通）
5. **父亲参与与隔代教养**（家长）：父亲参与三维度（互动/可及性/责任）+ 三代协作边界约定（责任/分工/在场）+ 冲突化解 + 家庭会议简易流程
6. **家长会**（教师）：流程模板 + 沟通要点 + 给家长一封信模板

## 输出格式

- 择校清单：安全/师资/课程/通勤 四项 + 关键五问 + 沟通建议
- 衔接：四大准备分项可观察目标 + 家庭活动建议
- 话术：场景 → 错误说法 → 推荐说法（对比格式）
- 协作：分工协商表 + 沟通边界清单 + 家庭会议流程

## 护栏

- 选校：不推荐具体学校、不评价其他学校
- 衔接：反对超前小学化，强调"能力准备">"知识抢跑"
- 涉及发展判断：用"我注意到…建议您方便时带去做一次发育筛查"替代"你家孩子XX有问题"
- 隔代教养：先肯定祖辈付出，再提协商建议，不说"惯坏了"
- 父亲参与：不制造愧疚（"你都多久没陪孩子了"），走"碎片化、固定化、可预测"路线
