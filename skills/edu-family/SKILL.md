# /edu-family 家庭日常管理

帮家长把家里的节奏理顺——作息、屏幕、规则、任务管理和学习规划，一个入口全管。面向 3-9 岁孩子家长。

执行前先读：knowledge/skill-packages/25-家长实务Tec.md（第二章「作息假期与屏幕规划」+ 第五章「家庭时间管理与任务执行」+ 第六章「家庭规则与陪伴质量」+ 第四章「学习与阅读规划」）。

## 做法

1. **作息与屏幕**：给可视化作息表模板 + 晨晚间固定流程 + 刚性边界/弹性空间划分。屏幕管理走"全家公约"路线——什么时候用/用多久/看什么/哪里不能用，孩子参与签字。加上幼小衔接四大准备提示
2. **时间与任务**：视觉计时器、任务拆解为小步骤、习惯绑定（"在已有习惯之后做新习惯"）、正向强化。番茄钟按年龄调整时长（5-6岁15分钟/7-8岁20分钟），不用手机计时
3. **家庭规则与陪伴**：3-5 条核心规则（安全/尊重/健康），和善而坚定执行，自然结果+逻辑结果。陪伴看质不看量——每天 10-15 分钟专属时间、孩子主导、全心投入、不教不评
4. **学习与阅读规划**：兴趣班取舍决策（观察孩子主动选择→广撒网→少数深耕）、分级阅读框架、对话式阅读提问示例

## 输出格式

- 作息：可视化一日表（时间段/内容/图标提示） + 固定流程 + 边界说明
- 屏幕：家庭媒体公约模板（可打印签字） + 替代活动清单
- 任务：可视化任务清单（图标+简短文字+打勾框） + 番茄钟按龄设定
- 规则：核心规则清单 + 执行一致口诀 + 逻辑后果示例
- 陪伴：每日专属时间安排 + 地板时间/黄金15分钟指南
- 规划：兴趣班决策清单 + 分龄分级阅读书单框架 + 亲子共读提问示例

## 护栏

- 作息不排到分钟，留弹性空间
- 屏幕不一刀切禁止，用"公约代替命令"
- 规则不靠吼——靠一致执行和家长的以身作则
- 不要求幼小衔接阶段刷题抢跑，重点在自理能力与任务意识
