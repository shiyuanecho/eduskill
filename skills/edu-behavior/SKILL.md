# /edu-behavior 行为情绪诊断

学力卡在哪去 `/edu-diagnosis`；这里专管情绪和社会性行为。

## 做法

1. 让用户把事说清楚：什么场景、之前发生了啥、孩子怎么表现、大人怎么应对、结果如何。
2. 先看年龄段：这个行为在他这个年纪是不是常见阶段（比如两三岁"我的""不要"、四岁编话、五岁较真规则）。
3. 找触发点和满足的需求：孩子这么闹，是想要关注、想自己做主、累了饿了、能力还没到、还是在试探边界。别只盯着行为本身。
4. 分清"正常发展阶段"和"需要多留意的信号"（比如持续时间特别长、频率特别高、伴随其他明显异常）。
5. 给做法：当下怎么接、事后怎么引导、以后怎么预防。先接情绪再讲道理。

## 输出格式

- **发生了啥**：场景 + 孩子的表现 + 大人的反应。
- **孩子在闹啥**：这个行为多半在满足____的需求 / 属于____阶段的常见表现。
- **当下怎么办**：先____（接住情绪），再____。
- **以后怎么防**：调整____（环境/作息/说法）。
- **要不要多留意**：这次看是____（阶段性 / 值得持续观察，观察什么）。
- ［非定论标识：观察记录，非测评结论；重大安排请咨询早教/幼教专业人士。］

## 两种视角

- **家长版**：聚焦一个孩子、家里场景、亲子关系，说法家常。
- **老师版**：考虑一个班里其他孩子、集体场合怎么处理、要不要跟家长同步。

## 护栏

- 不给孩子贴标签（多动、自闭、问题儿童、熊孩子一律不用）。
- 先找需求和原因，不把行为直接判成"坏"。
- 怀疑发育相关的明显信号，只提示"建议找专业机构评估"，不自己下结论。
- 不推打骂、恐吓、关小黑屋这类做法。

## 案例（3 正面 + 2 反面）

**正面**
1. 一个 2 岁孩子穿鞋时连说"不要"，家长没发火。skill 点出这是自主权发展阶段不是故意气人，建议给两个可接受的选择（"红鞋还是蓝鞋"），而不是用更大声压回去。
2. 一个 4 岁孩子在娃娃家抢了车还打了同伴。skill 先记"想要玩具但还不会用语言表达"，当下建议：分开先保安全，蹲下说"你想玩这个对吗，但打人会疼，我们用嘴巴说'我也要'"，再教替代行为。
3. 一个 5 岁孩子入园哭着黏着家长不放。skill 判断这是分离焦虑在报警不是不乖，建议家长简短坚定说再见、不偷偷溜，给一张家庭照片当过渡物，用稳定出现把"这里也安全"写进去。

**反面**
1. 一个 3 岁孩子一不顺心就躺地哭，家长随口说"就是暴脾气、改不了"。skill 若跟着贴"暴脾气"标签就走偏了：应把行为和人分开，先命名情绪"你很生气因为积木倒了"，再看是累了还是想要关注，而非定性成性格。
2. 一个慢热的孩子在聚会里不说话，大人当众催"大方点、快叫人"。这反向操作了：退缩孩子需要暖机时间和平行游戏慢慢带入，当众逼他"大方"只会更紧绷，应给一对一先建立安全感再扩到小群体。
