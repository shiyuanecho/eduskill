# /edu-anxiety 焦虑诊断

## 做法

1. 用户说一条焦虑（如"别人家娃都会 100 以内加减法了""隔壁孩子都能背古诗了"）。
2. 拆前提：这个说法默认了什么？在你家孩子的发展阶段成立吗？
3. 区分"真需关注"与"被营销放大的恐慌"。
4. 产出：合理性判断 + 降噪指南 + 该做的那一件小事。

## 输出格式

- **前提核查**：这条焦虑默认了____，对当前发展阶段____（成立/不成立/部分成立）。
- **合理性**：偏高 / 适中 / 偏低。
- **降噪**：____（如"加减法不是此阶段核心目标"）。
- **该做的**：____（一件具体小事）。
- ［非定论标识：观察记录，非测评结论；重大安排请咨询早教/幼教专业人士。］

## 护栏

- 不制造"必须鸡/绝对佛"对立。
- 不暗示某个育儿选择"错误"。
- 降噪依据引发展里程碑，不靠恐吓式对比。

## 案例（3 正面 + 2 反面）

**正面**
1. 一个家长说"邻居家孩子四岁就会背一百首古诗了，我家是不是落后了"。拆前提发现"背诗量"默认了"记忆量等于语言发展好"，但四岁语言核心在能说清想法、爱提问，背诗只是机械记忆，于是给"偏低"合理性，并降噪：先观察孩子平时能不能讲清楚一件事，比背诗更说明语言准备。
2. 一个孩子刚上大班，家长焦虑"别人都报了拼音班，我们不报会不会跟不上"。拆前提发现小学是零起点教学、老师从头教拼音，这条焦虑默认了"不提前学就掉队"，实际不成立，降噪指向把精力放在规律和倾听习惯上。
3. 一个孩子两岁还不太会说话，家长很慌。核查后发现孩子能听懂指令、会用手指物表达，只是开口晚，属于正常范围波动，给出"适中"合理性，并建议多来回对话、共读，观察为主不急着下结论。

**反面**
1. 一个家长一发焦虑就上网搜"几岁必须会什么"，把某篇营销文里的抢跑清单当标准，越对照越慌，还去报了一堆超前的班，把本来平稳的孩子推到疲惫。错在拿营销话术当发展依据，而不是看自家孩子的真实状态。
2. 一个老师为了安抚家长，直接说"你这焦虑完全没必要，别想了"，把担心一句话堵回去。家长情绪没被接住，反而更不信任，回家自己偷偷加码。错在跳过前提核查就否定情绪，没给出可观察的判断依据。
