# /edu-talk 亲子沟通与情绪引导

帮家长掌握"说人话"的沟通方法，以及孩子在情绪波动、大发雷霆、缺乏安全感时的引导策略。面向 0-9 岁孩子家长。

执行前先读：knowledge/skill-packages/25-家长实务Tec.md（第一章「亲子沟通与情绪引导」）和 knowledge/skill-packages/30-发展科学与脑科学.md（第四章「情绪智力与社会性发展」+ 第二章「脑科学与执行功能」）。

## 做法

1. **沟通话术工具箱**：
   - 我信息："当（事实）+我感到（感受）+因为（影响）+我希望（请求）"
   - 有限选择：两个都可接受的选项，减少权力斗争
   - 启发式提问："你觉得刚才发生了什么？你有什么办法解决？"
   - 描述性鼓励：夸"努力/策略/过程"（"我看到你试了好几种方法"），不夸天赋（"你真聪明"会让孩子回避挑战）
   - 避免贴标签："你总是""你真笨""你太粗心"会固化自我认知

2. **情绪教养**：
   - 情绪教练五步：觉察→视为亲近教导的机会→共情倾听并确认→帮孩子命名情绪→设限同时共同解决
   - 工具：情绪温度计（0-10，超7分先安抚不谈事）、情绪词汇卡（生气/失望/委屈/嫉妒/自豪等更细的词）
   - 关键时刻生理唤醒期不讲道理——杏仁核劫持前额叶，先安抚等平静

3. **安全感建设**：
   - 每天 10-15 分钟专属时间：三特征——每天发生、孩子主导、全心投入
   - 安全依恋不是时时抱着——是孩子需要时你在、能读懂信号、用他需要的方式回应
   - 分离焦虑是发育信号不是问题——建立"会回来"的预期，不骗、不突然消失
   - 高敏感不是难搞：可以调低刺激水平，但不改造孩子的气质

4. **孩子崩溃/发脾气的实操应对**：
   - 先做到自己的情绪暂停（嗓子升高/想数"123"=需要暂停的信号→去暂停角→呼吸平稳→再回来解决）
   - 幼儿崩溃五阶段应对 + 333法则：3 次深呼吸→3 个选择（有限）→3 分钟不讲话陪在旁边
   - 恢复后复盘用"我看到/我感觉/我们可以"结构

## 输出格式

- 话术：错误说法 → 推荐说法（对比格式）+ 原理一句话
- 情绪引导：场景（孩子怎么了）→ 分步回应（1-2-3）→ 可以说的原话
- 安全感：问题描述→判断（是不是安全感问题）→3 件可以做的事

## 护栏

- 不把"我信息"当控制手段（"我觉得你太自私了"="自私"是评判不是感受）
- 不使用奖励收买情绪（"别哭了给你买糖"——情绪被交易掉，孩子学不会调节）
- 不说"不许哭""别怕"等否定情绪的话——接住情绪不等于接受不当行为
