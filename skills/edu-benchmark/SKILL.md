# /edu-benchmark 对标

## 做法

1. 用户描述当前在做的启蒙内容 / 资源。
2. 对照发展核心维度（数学思维、语言沟通、社会性、运动、习惯、情绪管理等）逐项核查覆盖度。
3. 产出差距地图：已覆盖 / 薄弱 / 缺失。

## 输出格式

- **覆盖度**：维度清单 + 每维状态（已覆盖/薄弱/缺失）。
- **缺口**：最该补的是____。
- **建议**：先补____，再补____。
- ［非定论标识：观察记录，非测评结论；重大安排请咨询早教/幼教专业人士。］

## 护栏

- 对标的是"维度覆盖"，不是"比别人家快"。
- 不制造稀缺焦虑，缺什么就平实说明。
- 维度定义引《指南》与数学认知发展研究共识。

## 案例（3 正面 + 2 反面）

**正面**
1. 一个家长描述在家做的启蒙：每天亲子共读、数数、认字卡。对照维度核查，语言沟通和数感已覆盖，但运动和情绪管理几乎空白，产出差距地图：已覆盖两项、缺失两项，建议先补每天户外跑跳和情绪命名。
2. 一个机构想看自己的课程完整度，列出内容后逐项对照社会性、习惯、数学思维等维度，发现"习惯"只停留在上课守规矩、回家完全没衔接，于是把家庭延伸的小任务补进体系。
3. 一个老师接手新班，对照核心维度扫一遍，发现孩子整体语言不错但坚持性偏弱，就把"遇挫怎么办"作为本学期重点，而不是再多加一节知识课。

**反面**
1. 一个家长做对标时只数"今天认了几个字、算了几道题"，把覆盖度等同于知识量，忽略运动、社会性、情绪这些维度，最后体系严重偏科自己还以为很全。错在把对标当成"比别人快"的竞赛。
2. 一个机构拿对标结果吓家长："您家孩子缺失五六个维度，再不报班就晚了"。把平实的差距地图变成稀缺焦虑，家长慌着买单。错在制造恐慌而不是平实说明缺什么、先补什么。
