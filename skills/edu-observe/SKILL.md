# /edu-observe 观察评估模板库

观察评估不是"给幼儿打分"，而是幼儿教师最基本的专业工作方式。核心目的：看见真实的儿童、支持课程决策、实现家园共育与早期识别。美国幼教协会（NAEYC）明确评估应服务"改进教与学"，不是甄别或贴标签；教育部《3-6岁儿童学习与发展指南》（2012）反对"用一把尺子衡量所有幼儿"。方法可按"描述—取样—量化—诠释"分级，按目的选用，不必贪多。

## 做法

1. 先定观察目的：是记个别关键时刻、统计某行为频率、阶段性摸底，还是分析作品思维。
2. 选方法：日常管理用叙事记录+学习故事；研究特定行为用事件取样；阶段性建档用检核表；作品思维用作品分析。
3. 填对应模板，客观记录栏只写可验证事实，解释/推测单写并标注。
4. 同一行为至少观察3次、跨不同情境再判断，不一次定性。
5. 记录同时写进步与"能做的"，不把观察变监视；检核异常须转介专业机构，不自行诊断。

## 输出格式

- **方法选择速查**

| 级别 | 方法 | 适合 |
| --- | --- | --- |
| A 描述 | 叙事记录（轶事）、学习故事 | 日常、个性化记录 |
| B 取样 | 时间取样、事件取样（ABC） | 统计行为频率/分析触发情境 |
| C 量化 | 检核表、评定量表 | 筛查、对比、建档 |
| D 作品 | 作品分析 | 艺术/科学/语言等难量化领域 |

- **幼儿行为观察记录表模板**

| 项目 | 内容 |
| --- | --- |
| 幼儿姓名 / 代号 | （可用编号保护隐私） |
| 观察日期 / 时间 | 年 月 日 起止时段 |
| 观察情境 / 地点 | 如：自由活动·建构区 |
| 观察目的 | 如：了解其同伴交往方式 |
| 客观记录（我看到/听到的） | 按时间顺序写事实，原话加引号 |
| 我的解释（推测/分析） | 与"客观记录"分开书写，标注是推测 |
| 下一步支持策略（回应） | 具体、可操作 |

填写守则：客观记录栏只写可验证事实；解释栏可写但要标推测；同一行为至少观察3次再下结论。

- **发展检核清单（按五大领域，示例）**

说明：下表参照《3-6岁儿童学习与发展指南》五大领域典型表现的"示例性"条目，实际判定以官方《指南》原文为准（建议核查具体年龄段表述）。勾选栏填：已做到 / 发展中 / 暂未出现。

| 领域 | 观察要点（示例） | 3-4岁示例线索 | 4-5岁示例线索 | 5-6岁示例线索 | 勾选 |
| --- | --- | --- | --- | --- | --- |
| 健康 | 大动作/精细动作 | 能双脚灵活交替上下楼梯 | 能助跑跨跳过一定距离 | 能连续跳绳、手脚协调 | □ |
| 健康 | 生活习惯 | 在提醒下洗手、不偏食 | 主动刷牙、不偏食挑食 | 主动保护视力、按需饮水 | □ |
| 语言 | 倾听与表达 | 能听懂简短指令并回应 | 能基本完整讲述所见所闻 | 能有序、清楚讲述经历 | □ |
| 语言 | 阅读准备 | 喜欢听故事、翻书 | 喜欢涂涂画画表达意思 | 愿意用图画符号表达想法 | □ |
| 社会 | 人际交往 | 想加入同伴时敢提出 | 会轮流、商量解决冲突 | 能分工合作完成小组任务 | □ |
| 社会 | 社会适应 | 在提醒下遵守基本规则 | 能遵守规则不任性 | 遵守规则并理解行为后果 | □ |
| 科学 | 科学探究 | 对周围事物好奇、爱提问 | 能对事物比较、分类 | 能动手探究并记录发现 | □ |
| 科学 | 数学认知 | 能点数5以内物体 | 能按数取物、比较多少 | 理解10以内数量关系 | □ |
| 艺术 | 感受与欣赏 | 喜欢好看、好听的事物 | 能专心欣赏并表达感受 | 能多角度感受自然与艺术作品 | □ |
| 艺术 | 表现与创造 | 能模仿唱跳、随意涂画 | 能用简单方式表现事物 | 能用多种材料大胆创作 | □ |

注意：此表为发展线索示例，不可当诊断标准；某领域多项"暂未出现"且明显偏离同龄，应结合 CDC 里程碑/ASQ 进一步观察并建议转介。

- **学习故事写作模板（注意—识别—回应）**（新西兰 Margaret Carr 基于课程 Te Whariki 提出，Carr 2001；用"你"称呼，聚焦"能做的"）

- **注意（故事+照片）**：今天在（情境），我注意到你（具体做了什么、说了什么）。
- **识别（学习品质）**：这展现了你的（如：坚持/好奇/责任/合作/自信）……因为（证据）。
- **回应（下一步支持）**：接下来我们准备（具体策略/环境调整/材料投放），帮助你继续（延伸方向）。

写作要点：用"你"称呼幼儿；聚焦"能做的"而非"缺的"；每个学习故事配一张真实照片或作品；结尾给家长一句可带回去的"发现"。

- **作品分析表**

| 项目 | 内容 |
| --- | --- |
| 作品类型 | 绘画/建构/记录表/手工 等 |
| 创作日期/情境 | 主题活动时/自由活动 |
| 幼儿原话（记录其说明） | 原话加引号 |
| 我观察到的能力与思维 | 如：能表现前后空间、会用符号代表数量 |
| 反映的发展领域 | 对应五大领域哪一点 |
| 下一步支持 | 投放什么材料/提什么问题 |

- **取样法记录要点**
  - 时间取样：把一段时间切固定间隔（如每5分钟），只在间隔点记目标行为"有/无"。适合统计频率（离座、同伴互动），会丢细节。
  - 事件取样（ABC）：预先定义行为（争抢玩具/求助），每次完整记前因A—行为B—后果C。适合分析功能与触发情境。

## 两种视角

- **老师版**：在园用观察记录支撑课程调整与家园沟通。日常管理以叙事+学习故事为主，阶段建档用检核表，研究特定行为用事件取样，筛查预警参考 CDC 里程碑/ASQ（须专业人员规范使用）。
- **家长版**：在家也能用简化观察——记孩子"今天在XX情境做了什么、说了什么原话"，不写"他胆小/笨"；同一表现跨几天、不同场景看，再判断。拿不准时把记录带给老师或专业机构，不当自己下结论。

## 护栏

- 只做观察记录，不下儿童能力判定结论。
- 凡涉及儿童发展的结论，尾部带非定论标识：「观察记录，非测评结论；重大安排请咨询早教/幼教专业人士」。
- 知识引权威（发展指南、课标、发展心理学共识），不编造来源；拿不准的书名/研究/数据如实说无法核实。
- 说人话：大白话，像中国人正常说话，别用翻译腔和学术黑话；不加表情符号。
- 去冲突化：不制造对立、不煽动焦虑、不暗示某个选择"错了"。
- 涉及发育/行为/医疗/诊断的内容，标注「筛查非诊断、疑点转介专业」，不自行判定。

## 案例（3 正面 + 2 反面）

**正面**
1. 中班男孩小宇（化名）集体活动极少发言，教师起初写"性格内向、不爱表达"让家长焦虑。改用事件取样连续一周记录：建构区与同伴合作会用短句；集体面前发言近零；一对一能答简单问题。客观记录写"区域活动时，小宇对同伴说'把这个放上面'并递出积木"，用检核表发现其表达理解正常，仅"集体面前讲述"偏弱。写学习故事聚焦"用行动表达、在熟悉伙伴前愿沟通"，先从2-3人小组讲述过渡。把标签还原成情境行为。观察记录，非测评结论；重大安排请咨询早教/幼教专业人士。
2. 大班小杰常在争抢材料时推搡。教师用 ABC 事件取样记5次：前因（材料被拿/排队被插）、行为（推人）、后果（得材料/被制止），发现4次前因是"想要材料但不会用语言表达"。回应：教"我可以一起玩吗"、材料区加共享规则图示；两周后推搡明显下降。事件取样区分了"行为表象"与"真实功能"。筛查非诊断、疑点转介专业。
3. 一位教师做作品分析：收下孩子画的"我的家"，记原话"这是妈妈，外面有太阳"，识别出其能表现"空间前后+用符号代表人物"，下一步投放更多低结构材料鼓励讲述。记录含进步与思维，非只记缺点，观察变支持而非监视。

**反面**
1. 有教师没观察几次就用评定量表给孩子打"1-5分"。走偏了：凭印象打分主观偏差最大，应以多次、多情境客观记录为依据，量表仅辅助且注明证据来源。
2. 一位老师把检核表多项"暂未出现"直接写成"发育迟缓"。走偏了：检核/筛查只是预警信号，异常须转介专业机构（如妇幼保健院、儿童保健科）做正式评估，不能把筛查当诊断。筛查非诊断、疑点转介专业。
