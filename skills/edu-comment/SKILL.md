# /edu-comment 写评语

把你对孩子的观察，变成家长愿意反复读、孩子能建立自我认知的一段话。不是打分，是让家长看见孩子真实的发展过程。

## 做法

1. **先收素材**：你手头有什么观察？一次具体的事（哪天、什么活动、孩子怎么做的）、作品、一句话原话都算。没有就先问用户要一段真实观察，别凭空编。
2. **挑标尺**：看这个孩子属于哪种——多数孩子用"看见-描述-期待"三段；想突出学习品质的用学习故事（注意-识别-回应）；发展慢一点的用鼓励型（落点在"比之前进步了什么"）；明显超前的用挑战型（给下一步可探索的方向）。
3. **写具体行为，不写标签**：说清"他做了什么、用了什么办法、有个什么细节"，别写"聪明""调皮""内向"。夸努力和策略，不夸天赋。
4. **给能落地的下一步**：家长读完得知道"在家可以怎么陪"，孩子读完知道"我下次可以试试什么"。
5. **检查**：对照文末清单，通篇没有空泛形容词、没有贴标签、没有只写优点或只写缺点。

## 输出格式

### 模板A：发展性评语"看见-描述-期待"三段式

- 看见：我注意到你最近在（具体情境/活动）中……
- 描述：你（具体行为 + 努力/策略），比如（一个真实细节）。
- 期待：接下来，我们可以一起试试（贴合其水平的下一步）。

### 模板B：学习故事（Learning Story）记录

- 注意（故事）：那天发生了什么，你做了什么。
- 识别：这展现了你的（学习品质，如坚持、勇气、责任）。
- 回应：我们准备怎样支持你继续探索。

### 模板C：差异化分级话术库

- 鼓励型（发展慢一点）："这学期你能自己穿好外套再出门，比开学时快了很多，真为你高兴。"
- 挑战型（明显超前）："你对恐龙的分类已经讲得很有条理，下次可以试着做一本'恐龙小百科'图画书，把你的发现记下来。"
- 标准型（多数孩子）：用模板A，具体行为 + 一个细节 + 一个下一步。

### 检查清单

- [ ] 有没有具体行为，而非"聪明/调皮/内向"等空泛词？
- [ ] 有没有聚焦"能做的"，而非挑缺？
- [ ] 有没有贴合这个孩子真实的水平与发展阶段？
- [ ] 家长读完能否知道"下一步可以怎么支持"？

［非定论标识：观察记录，非测评结论；重大安排请咨询早教/幼教专业人士。］

## 两种视角

### 老师版（写给孩子 / 写给孩子家长）

- 学期评语：一个人一段，个性化，不套"你真棒"。落点在行为和孩子能做的下一步。
- 家园联系册一句话：一句具体观察 + 一句小期待，方便家长接话。例："这周建构区他主动教同伴用拱形门更牢固，在家可以让他讲讲'什么搭得稳'。"
- 学习故事：图文记录"魔法时刻"，用第二人称"你"，把孩子放在主角位置。
- 差异化：发展慢的表扬小步进步，超前的给挑战方向，避免千篇一律。

### 家长版（写给孩子老师的反馈）

- 目的：把在家观察到的具体变化，简洁地反馈给老师，帮老师更了解孩子，也让家园目标对上。
- 写法：先说一件具体的事（时间/场景/孩子怎么做的），再表达一个想和老师配合的小希望。不诉苦、不挑刺、不比较别的孩子。
- 例："老师您好，最近在家他愿意自己收碗筷了，虽然还慢。我们想和您配合，在园也多给点'自己的事自己做'的小任务，您看合适吗？"

## 护栏

- 只做观察记录，不下儿童能力判定结论。
- 凡涉及儿童发展的结论，尾部带非定论标识：「观察记录，非测评结论；重大安排请咨询早教/幼教专业人士」。
- 知识引权威（发展指南、课标、发展心理学共识），不编造来源；拿不准的书名/研究/数据如实说无法核实。
- 说人话：大白话，像中国人正常说话，别用翻译腔和学术黑话；不加表情符号。
- 去冲突化：不制造对立、不煽动焦虑、不暗示某个选择"错了"。
- 涉及发育/行为/医疗/诊断的内容，标注「筛查非诊断、疑点转介专业」，不自行判定。

## 案例（3 正面 + 2 反面）

**正面**
1. 一位老师给中班小明写学期评语，没有写"表现很好很聪明就是有点调皮"，而是："我注意到建构区里，你会用圆柱体积木搭出带旋转楼梯的城堡（看见）。你还教同伴'用拱形门更牢固'，愿意把办法分享给别人（描述+细节）。下学期我们可以一起画一张'建筑物稳固设计图'，把你发现的办法用到更大的项目里（期待，挑战型）。"把标签换成了可观察行为和可操作的下一步。
2. 一位老师用学习故事写一个小女孩："注意——今天你蹲在角落，用半小时把散落的积木按颜色分成了五堆，还数了每堆有几个。识别——这展现了你的耐心和会自己想办法。回应——我们准备在科学区多放些可以分类的自然物，让你带着小朋友一起玩分类游戏。"第二人称、孩子当主角，家长读完看得见孩子的成长。
3. 一位家长给孩子老师写反馈："老师您好，最近在家吃饭他能自己拿勺子、撒了也自己擦，比上个月愿意多了。我们想和您配合，在园也多让他自己动手做力所能及的事，您看行吗？"具体、不挑刺、提出可协作的小步，老师顺势给了配合建议。
4. 一位老师给发展慢一点的孩子写："这学期你能自己穿好外套再出门，比开学时快了很多，真为你高兴。"落点在小步进步，不拿他和别人比，也没贴"慢"的标签。

**反面**
1. 一位老师全班写"你真棒，表现优秀，继续加油"。这走偏了：千篇一律、空泛表扬，家长看不出孩子到底是怎样的学习者，家园沟通价值为零。应每人写一件具体行为和对应的下一步。
2. 一位老师写"这孩子太内向，不爱说话，要多练社交"。这走偏了：用特质词贴标签，会固化孩子自我认知，家长也容易挂不住面子。应改成可观察描述，比如"在小组活动时他更多在听同伴说，这周有两次主动递了材料给小朋友，我们可以一起多创造他愿意开口的小机会"。
3. 一位家长给老师写"您多管管我家孩子，他在家皮得不行，肯定是班里最闹的"。这走偏了：把在家的不满甩给老师、还拿孩子和别人比，制造对立。应改成一件具体观察 + 一个想一起配合的小希望，不比较、不问责。
