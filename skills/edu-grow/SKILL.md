# /edu-grow 发展科学与脑科学

不是一个独立工具，是给所有 edu Skill 当"原理层"的。老师或家长拿到方法后想知道"为什么这个方法有效"，或遇到看不懂的孩子行为想了解背后的科学——来这里。面向教师和家长。

执行前先读：knowledge/skill-packages/30-发展科学与脑科学.md。

## 做法

1. 确认用户的疑问属于哪个领域：发展理论/脑科学/学习科学/情绪
2. **核心发展理论**：
   - 皮亚杰：解释"孩子为什么转不过弯"——前运算阶段还不懂守恒
   - 维果茨基：解释"什么时候教最有效"——瞄准最近发展区（跳一跳够得着）
   - 依恋理论：解释"为什么安全感是所有学习的前提"——安全基地允许向外探索
   - 敏感期：解释"为什么孩子会突然对某件事痴迷"——不是执拗是发展的节奏
3. **脑科学**：
   - 执行功能发展：解释"为什么孩子管不住自己"——前额叶远未成熟
   - 情绪神经基础：解释"为什么孩子崩溃时讲道理没用"——杏仁核劫持前额叶
   - 运动/睡眠/营养对大脑的影响——用"大脑肥料/记忆存档/代谢清洁"等通俗比喻
4. **学习科学**：
   - 成长型思维 vs 固定型思维：解释夸"努力"和夸"聪明"的天壤之别
   - 间隔练习与检索练习：解释为什么分散学比一口气学更好
   - 高质量的玩 = 最高级的学习：解释为什么不能把玩和学对立
5. **给出"可以这样跟家长/孩子/同事解释"的话术模板**——不说术语不装神弄鬼

## 输出格式

- 场景→行为→科学解释（1-2 段，通俗易懂）→"可以这样说"（原话模板）→相关概念延伸
- 对话体："孩子为什么会……"→ 一段通俗解释 → "下次看到这个行为，你可以……"

## 护栏

- 只讲科学，不取代专业判断
- 通识性科普，不编造研究数据
- 发展理论是解释框架不是铁律——个体差异大
- 不用于商业营销话术
- 引用的理论和研究以公认主流共识为准，边缘/争议观点标注"有争议"
