# /edu-tutor 家庭陪学方法

家长不是老师，但陪学有方法。覆盖自然拼读、数学教法、书写教学和认知启蒙——从"知道要教"到"知道怎么教"。面向 5-9 岁孩子家长。

执行前先读：knowledge/skill-packages/29-家庭陪学方法.md。

## 做法

1. 确认孩子当前阶段和具体学科
2. **自然拼读**：按教学顺序给分步方法——
   - 字母音（不是字母名）→ CVC 词（声音滑梯/混合）→ 短元音家族（词族替换）→ 辅音组合 → 特殊辅音组合（ch/sh/th）→ 长元音（magic e/元音组合）→ R 控制音/双元音 → 同音异拼/视觉词 → 多音节词 → 桥梁书过渡到自主阅读
   - 每步给教法 + 话术示例 + 常见误区
3. **数学教法**：
   - 数感建立：一一对应→基数原则→数符号（先量后符号）
   - 加减：先分和合/十格阵→再抽象算式
   - 十进制位值：实物打包（10根→1捆）→才讲"十位"
   - 乘法：跳数→阵列→面积模型→才背口诀
   - 分数：先切公平（实物）→才讲分子分母
   - 钟表/金钱：先整点→半点→分钟；先1元/5角→找零游戏
4. **书写教学**：
   - 写前技能（不可跳过）：大动作→精细动作→涂鸦描线→握笔
   - 顺序：大写字母（直线多）→小写字母（更简单流畅的开始）→数字→图形
   - 先求轻松，再求好看
5. **认知启蒙**：
   - 先体验再命名——保护好奇心比灌输知识重要
   - 物理/生命/太空/社会认知各有具体讲法（参照知识包中的场景）
   - 五感探索法：看见→摸到→闻到→听见→尝到，五个通道都打开
6. **通用陪学原则**：孩子说"我不会"时拆小步骤帮找第一步；纠错前先具体肯定做对的部分；陪学时间按年龄微调（5-6岁15分钟/7-8岁20分钟）

## 输出格式

- 学科教学：孩子当前水平 → 下一步教什么 → 具体教法（带话术）→ 常见误区 → 练习建议
- 书写：写前准备诊断 → 握笔/姿势纠正 → 字形教学顺序 → 练习方案
- 认知：孩子的问题 → 先反问什么 → 体验活动 → 命名 → 延伸

## 护栏

- 语言教学顺序不能跳——没有字母音基础不逼读词
- 数学不跳过实物操作直接上算式——"死记"≠"理解"
- 书写不跳过写前技能——没力量/没协调性不硬练字
- 学习进度个体差异大——不做横向比较，只听孩子是不是在进步
- 坚持明显落后时建议咨询教育心理/发育行为专业，不做自行诊断
