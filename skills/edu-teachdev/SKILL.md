# /edu-teachdev 教师专业成长

从"会上课"到"能带团队"的教师发展支持。覆盖教研、反思写作、培训和家长讲座。面向幼儿园和小学低年级教师。

执行前先读：knowledge/skill-packages/26-教材书摘Tec上.md（「教师专业成长与教研」章节）和 knowledge/skill-packages/24-教师实务Tec.md（相关章节）。

## 做法

1. **教研活动策划与实施**：
   - 像备课一样备教研：目标→问题→流程→产出
   - 七种教研形式菜单：案例式/解惑式/观摩式/沙龙式/网络式/诊断式/辩论式——按目的选最合适的
   - 教研循环流程：问题→方案→实施→反思→总结→新问题
   - 组织者核心能力：归纳→演绎→链接→迁移

2. **教学反思与写作**：
   - 反思五类写法：教育日志/教育叙事/教育案例/教育反思/教育论文
   - 叙事骨架：情境→行为→分析→改进
   - 每周反思清单："今天我尊重了孩子的发展需求吗？我的干预是支持还是控制？我的教学匹配孩子的发展阶段吗？"

3. **与专业标准对齐**：
   - 对照《幼儿园教师专业标准》三维度十四领域：专业理念与师德/专业知识/专业能力
   - 教师自评可用这十四个领域做对照表，标记"做得好/可提升/需学习"
   - 专业能力维度的核心关注项：环境创设与利用、一日生活组织与保育、游戏支持与引导、教育活动的计划与实施、激励与评价、沟通与合作、反思与发展

4. **家长课堂/讲座素材**：
   - 主题框架→PPT 大纲→讲稿→互动环节→一页纸备忘
   - 面向家长：不说术语，用生活中的例子
   - 可以配上"为什么这个方法是有效的"简短科普（调用 edu-grow 的发展科学内容）

## 输出格式

- 教研：策划卡（目标/问题/流程/产出）+ 形式决策表
- 反思：事件→我的做法→效果→如果重来会怎样→下周改什么
- 培训：主题→目标→内容（3个关键点）→互动→材料→评估
- 讲座：主题→PPT大纲→讲稿开篇→主要论点→互动→一页纸

## 护栏

- 教研不脱离真实课堂问题——从观察中来，回到实践中去
- 反思不写"反省书"式的自我批评——聚焦"学到了什么、下周改什么"
- 培训内容与教师当前能力水平匹配——不超纲不务虚
- 不制造"好老师"vs"差老师"的二元对立——聚焦成长而非评判
