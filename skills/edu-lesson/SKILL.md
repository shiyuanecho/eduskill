# /edu-lesson 教案与活动设计

把"想让孩子获得什么"变成明天就能拿去用的教案和活动。覆盖幼儿园到小学低年级（3-9 岁），老师和家长都给。

执行前先读：knowledge/skill-packages/26-教材书摘Tec上.md（备课与活动设计相关章节）和 knowledge/skill-packages/24-教师实务Tec.md（第一章「备课与教案设计」）。

## 做法

1. 确认目标任务和孩子年龄段
2. 按以下维度组织输出：
   - **教案/活动方案**：主题/目标（对照《指南》五大领域或课标）→ 材料准备 → 环节流程 → 分层提示 → 弹性预案
   - **概念讲法**：把抽象概念（数感/时间/分类/因果等）用孩子能懂的话、道具和比喻讲出来——先实物操作，再用图，最后才抽象
   - **提问设计**：围绕目标设计开放式+封闭式提问组合，指向思考而非记忆
   - **教具与材料**：优先低结构（积木/自然物/废旧材料）、可重复、让孩子自己"发现"对错的
   - **环境布置**：区角怎么分、材料怎么投、主题墙怎么做、过渡怎么衔接
3. 给"家长在家也能做"的简化版（一项核心活动 + 一句话提示）

## 输出格式

- 教案：设计意图 → 目标（可观察动词）→ 准备 → 环节（导入/探究/实践/总结，预估时间）→ 分层提示 → 弹性预案
- 概念讲法：孩子现有认知描述→用什么比喻/道具→分步引导（先动手→再归纳→最后给命名）→常见误区和纠正
- 提问清单：每个环节 2-3 个关键问题，标明是开放式还是封闭式

## 护栏

- 反对小学化——3-6 岁不教符号算式，用实物和游戏替代
- 目标用可观察行为动词（"能按大小排列 5 个瓶盖"），不写"培养兴趣"这种空话
- 别把环境当装饰——环境是"第三位教师"，区角划分决定行为，材料投放决定学习深度
- 技能类标准须与孩子年龄/发展阶段匹配（可参考 knowledge/skill-packages/30-发展科学与脑科学.md）
