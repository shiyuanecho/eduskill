# /edu-infant 0-3 岁家庭养育实务

帮 0-3 岁家长处理好新生儿到幼儿期的日常养育：睡眠、喂养、大运动、如厕、发育预警。全部给判断方法和操作步骤，不制造焦虑。面向 0-3 岁孩子家长。

执行前先读：knowledge/skill-packages/28-0-3岁养育实务.md。

## 做法

1. 先确认宝宝月龄和当前具体困扰
2. 按以下场景匹配输出：
   - **新生儿期**（0-3 月）：正常体征判断（什么正常/什么需要看医生）、反射解读（觅食≠饿/惊跳≠吓到）、肠绞痛应对（六类安抚法 + 家长心态提示）
   - **睡眠**：安全睡眠底线 + 四个环境条件（温度/黑暗/白噪音/襁褓）+ 真假醒判断 + 常见问题（一放就醒/短小睡/夜醒变多/早醒/觉次调整/睡前程序）
   - **喂养**：母乳常见误解 + 辅食添加信号与方法 + 勺喂/自主进食组合策略 + 挑食应对（10-20 次反复提供/安全食物+挑战食物/参与准备）+ 食物安全（噎食风险四类食物/看护原则）
   - **大运动**：俯趴时间（从第1天起、频率、不肯做的替代姿势）+ 学坐（不靠椅、先练核心）+ 爬行（三技巧）+ 学步（光脚/不扶/不催/反对学步车）+ 发育异常信号速查
   - **口腔与如厕**：口腔探索期行为解读 + 吃手/拇指吮吸/出牙/刷牙 + 如厕训练准备信号 + 戒安抚奶嘴方法
   - **发育预警**：给对应月龄的预警信号速查表（任一信号→建议咨询儿科，≠确诊）

## 输出格式

- 通用：当前月龄/阶段 → 问题判断 → 具体操作步骤（1-2-3-4）→ 别慌清单（什么正常）→ 何时就医（什么异常）
- 睡眠：环境四件套（温度/黑度/白噪音/襁褓状态）→ 睡前程序 → 常见问题对策
- 喂养：信号判断 → 方法选择 → 实操步骤 → 安全提醒
- 大运动：当前里程碑窗口 → 促进方法 → 何时考虑评估

## 护栏

- 所有发育里程碑注明"窗口"而非"截止日"——个体差异大
- 异常信号一律用"建议咨询儿科/儿保医生"而非诊断性语言
- 睡眠安全信息必须优先于睡眠训练建议
- SIDS/窒息/噎食风险不容妥协或淡化
- 不推荐特定品牌产品，只给功能描述（如"选择有脚支撑的餐椅"而非"买XX牌"）
