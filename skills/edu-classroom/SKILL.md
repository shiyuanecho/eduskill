# /edu-classroom 班级与课堂管理

日常带班中的秩序建立、安全应急、常规培养和差异化引导。面向幼儿园和小学低年级教师。

执行前先读：knowledge/skill-packages/24-教师实务Tec.md（第三章「差异化与个别引导」+ 第四章「不同特质儿童引导」+ 第九章「课堂突发与安全应急」）。

## 做法

1. 区分用户场景：是预防（建常规/设计分层）还是应急（冲突/受伤/突发）
2. **安全应急**（最优先）：按"先止伤→再安抚→后处理→最后沟通家长"顺序。给具体步骤，不是泛泛原则。涉及生理急症时强调教师不作医疗处置，第一时间按预案处置+联系校医/120
3. **常规建立**：给 3-5 条正面表述的规则 + 视觉提示 + 一贯执行 + 过渡环节用儿歌/音乐/游戏化指令
4. **差异化与分层**：流动分组、同目标不同路径、气质/学习风格适配。给分层任务卡模板。不贴"永远落后"的标签
5. **不同特质儿童引导**：攻击性行为（PBS正向行为支持）、退缩内向（渐进scaffold）、多动/冲动（结构化+视觉时间表+运动）、规则意识弱（体验自然后果）

## 输出格式

- 应急：场景→立即行动（1-2-3步骤）→后续处理→对家长话术
- 常规：目标行为→正面表述→视觉提示方案→过渡环节设计
- 分层：基础层/标准层/挑战层的活动变体 + 配对策略
- 特质引导：行为描述→功能分析→替代行为→环境调整→即时具体表扬

## 护栏

- 安全场景：不做医学诊断/医疗处置，明确转介专业
- 行为场景：只做观察记录不下能力判定；不贴"坏孩子""有问题"等标签
- 分层场景：不固化分组，保留上下通道
- 多动/ADHD：不自行诊断，只做教育观察，疑点建议专业评估
