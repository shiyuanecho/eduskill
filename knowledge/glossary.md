# 高频概念词典

本词典收录儿童成长 / 启蒙 / 教学领域的高频概念，供各子技能在讲到某概念时快速对齐含义，避免不同 skill 各说各话。

> 当前为框架版：词条先以「索引」形式列出，定义与常见误解从 `knowledge/skill-packages/` 与 `knowledge/atoms/` 提炼回填，不凭空编写。

## 按主题的概念索引

- **发展基础**：数感、执行功能、心理理论、依恋、游戏价值、内在动机、成长型思维
- **学习维度**：一一对应、数量保留、模式、空间几何、对话式阅读、汉字认知边界
- **语言与认知**：口语发展、高质量输入、认知真相、假装游戏、绘画阶段
- **社会情绪**：情绪安全、冲突解决、分离焦虑、行为解码、情绪调节
- **教学实务**：低结构材料、学习故事、区角、一日流程、过渡环节、家园沟通
- **衔接与体系**：幼小衔接、四大准备、体系覆盖度、对标差距地图
- **工具约定**：非定论标识（观察记录，非测评结论）、双视角（家长版 / 老师版）

## 词条待填格式

每个词条后续补：一句话定义 + 常见误解 + 对应 `skill-packages/` 文件 + 相关原子 id。

例：
- **数感**：对数字大小、数量关系、数量的直观感觉，不等于会数数。常见误解：把「能背数到 100」当成数感好。见 `skill-packages/04-数学启蒙.md`。
