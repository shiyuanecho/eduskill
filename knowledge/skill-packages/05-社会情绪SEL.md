# 社会性与情绪能力（SEL）领域参考

面向家长、幼儿教师及教育者助手。内容为社会情绪能力培养的专业共识，聚焦：情绪识别与表达的发展、情绪安全的探究环境、常见情绪行为问题的应对原则、SEL 典型发展顺序、对话式阅读促进社会情绪。

---

## 一、SEL 是什么：五维能力

社会情绪能力指一个人认识自己情绪、管理自己、理解他人、与人相处、并为行为负责的一整套能力。常被归纳为五个相互联系的维度：

- **自我觉察**：认得出"我现在是什么感受"，能给情绪起名字。
- **自我管理**：情绪上头时能用办法让自己平静，而不是立刻炸毛或崩溃；能等待、能坚持。
- **社会觉察（共情）**：看出别人的情绪，理解别人和我想的不一样。
- **人际关系技能**：会交朋友、会表达、会商量、冲突了能一起解决、困难时会求助。
- **负责任的决定**：遇到事能选一个不伤人也不亏待自己的做法，会想"这样安全吗、公平吗、后果是什么"。

五个维度拧在一起：能认出自己生气（自我觉察），才谈得上管住不踢人（自我管理）；能看见别人难过（社会觉察），关系行为（递纸巾、一起玩）才做得出来。

## 二、情绪识别与表达的发展顺序

孩子先能理解"别人的情绪"，再慢慢理解"抽象的情绪规则"，最后才学会调节自己。典型链路：

1. **识别情绪**：能在故事/画面里指认"小熊生气了"。
2. **给情绪起名**：把自己体验对上词，"原来我刚才也是生气了"。
3. **把故事里的情绪搬到自己身上**：理解自己当下的感受。
4. **学着调节**：生气时可以怎么做。

年龄段参考（2—6 岁普遍趋势，个体差异很大）：

- **2—3 岁**：能感知并表达基本情绪（开心/生气/害怕/难过），但大多用行为而非语言，如哭闹、肢体动作。
- **3—4 岁**：能命名基本情绪；开始从表情、动作线索判断情绪；可引导说"我生气了"。
- **4—5 岁**：观点采择萌芽，能想"他为什么这么做"；情绪词汇变细（失望、委屈、嫉妒）；冲突中能听简单道理。
- **5—6 岁**：能说出自已的调节办法（深呼吸、先离开）；能讨论规则和多种解法；开始为行为想后果。

关键原则：情绪得先被"命名"，才能被"调节"。永远不要在命名情绪之前，先要求消灭情绪。

## 三、情绪安全的探究环境怎么营造

孩子敢不敢探索、提问、试错，取决于这里犯错了会不会被笑。情绪不安全时，孩子大脑把资源拿去"自我保护"，挤占探索与思考的力气；他最先学会的是"别出头、别答错、别惹事"。

营造原则（低威胁、高支持）：

- **容错文化**：出错正常化为学习的一部分。积木倒了，第一反应是"哦，它倒了，我们看看为什么"，而不是"叫你小心"。
- **少评价人、多描述事**：不说"你真聪明"（易让孩子回避挑战），改说"你刚才试了三种办法让桥站稳"。描述努力和策略，更助于保持投入。
- **敏感回应**：孩子发出信号（指一下、问一句、哭一声），及时、一致、恰当地接住。蹲下平视、命名他的情绪、冲突里先连接情绪再引导行为。
- **可预测的安全基地**：稳定作息、清晰规则、可预期的过渡，降低不确定性带来的紧绷。每天用一样的流程和预告语，孩子知道下一步是什么。
- **允许"不知道"**：老师以共同探究者身份加入，示范"我也不会，我们一起查"。

场景抓手：
- 集体活动：允许非语言参与（点头、指一指），多开放式提问，允许说"不知道"。
- 探究活动：老师一起当探究者，把"不知道"变常态。
- 过渡环节：用预告和给选择（"再玩两分钟，你想先收积木还是先收书"）减少失控感。

判断情绪安全建起来的信号（别看"吵不闹"，看这几条）：
1. 出错或失败后，继续试还是立刻放弃、躲闪；
2. 愿不愿说"我不会""我不想"，而不是一味迎合；
3. 大人靠近时是放松还是紧绷；
4. 集体里敢不敢提问、敢不敢提不同意见。

注意：安静不等于安全。"安静听话"可能是"恐惧性顺从"——区别看孩子出错后的反应。

## 四、常见情绪行为问题：先接情绪，再引导

三类最常见信号：焦虑（分离焦虑等）、攻击（打人、抢东西）、退缩（不主动、不说话）。它们都不是"坏孩子"的证据，而是孩子还不会用更成熟方式表达困难的信号——像咳嗽发烧一样在提醒"内在撑不住了，需要被看懂、接住、帮忙"。

应对总顺序：大人先稳住自己 → 帮孩子说出情绪名字 → 给替代行为（不只说"不许"） → 先接住需求再教规矩 → 调环境减少触发 → 平时用故事/角色扮演练 → 多看做对的时刻、及时肯定 → 家校一致。

具体做法：

- **大人先稳自己**：孩子炸时你先深呼吸，心里想"他又遇到困难了，不是针对我"。你的平静就是孩子的定海神针。
- **给情绪命名（最省力也最管用）**："你很生气，因为积木倒了""你有点怕，因为妈妈要走了"。命名不等于纵容行为——"你生气可以，但不能咬人"，情绪被接住，行为才有商量余地。
- **给替代行为，而不只说不许**：生气了可以跺脚、抱枕头、来拉老师手；想要玩具可以指、可以问"轮到我吗"；怕分离可以带妈妈的小物件、约定"放学第一个接你"。替代行为要具体、可操作、当场示范。
- **先接住需求，再教规矩**：抢东西打人，先分开、确认都没伤着，再低声说"你想玩这个对吗？但咬人会疼，我们用嘴巴说'我也要'"。语气温和但坚定。
- **调环境**：玩具太少就加、空间太挤就分区、日程固定节奏，很多攻击和焦虑会自己降。

三类针对性技巧：

- **焦虑**：可预测的日常（"吃完点心我们就去操场"）；分离时简短坚定说再见、不偷偷溜、不拖泥带水；给过渡物（家庭照片、小玩偶）；提前预告变化；少吓唬。
- **攻击**：平静时教"手是交朋友不是打人"；冲突现场先保安全再讲；多安排等待和轮流的游戏练自控；观察触发点（累？饿？人太多？）提前避开；大人自己少用打骂做示范。
- **退缩**：别当众逼说，给"暖机"时间；从平行游戏开始（他在旁边看，你玩给他看，慢慢邀他加入）；用他擅长的事带他出场；一对一先建立安全感，再扩到小群体；多等几秒，给开口空间。

常见误区（家长老师都易踩）：
- 把信号当"品质"贴标签（"就是胆小""暴脾气"）——把行为和人分开。
- 只用惩罚不问原因——惩罚能停行为，教不出替代行为。
- 忽视或嘲讽焦虑（"这有什么好怕的"）——只会使焦虑更重。
- 逼退缩的孩子"大方点"——是反向操作。
- 把偶尔行为当"问题儿童"——看频率、强度、持续时间，不是看"有没有发生过"。
- 用哄骗和威胁交替（"你再闹我就走了"）——破坏信任，教交易代替表达。
- 把"不麻烦"当"健康"——过度乖、完全没自己声音，可能只是把困难咽下去了，同样要被看见。

## 五、对话式阅读促进社会情绪

共读（成人和孩子一起读，不是丢给点读笔）是培养 SEL 的天然练习场：氛围安全，孩子敢说"这页让我害怕"；绘本把抽象情绪具象成有脸有故事的角色；故事冲突天然提供"他为什么这么做"的提问机会。

操作框架（可上手，不必死守）：

- **PEER 四步**：提问（Prompt）→ 反馈/评价（Evaluate，接纳并扩展，不是打分评判）→ 扩展（Expand，把孩子的回答说得更丰富）→ 重复（Repeat，带孩子把扩展后的话再说一遍）。
  - 例：指图问"小熊怎么了"（P）→ 孩子答"生气"（E，点头"对"）→ "对，他气得脸都红了，这叫非常生气"（Ex）→ 带孩子把这句再说一遍（R）。
- **CROWD 五类提示**：填空式、回忆式、开放式、Wh-提问（什么/哪/谁/为什么）、联系生活（Distancing）。其中"联系生活"最关键。

对话要点：

- **给情绪起名字**："小熊现在是什么感觉？你从画面上怎么看出来的？"引导孩子从表情、动作判断。
- **把情绪说细**：别只停留在开心/不开心，引入失望、委屈、嫉妒、骄傲等更细的词。"对，他等了好久没人来，这叫失望，还有点委屈。"
- **开放式代替是非问**：少问"他生气了吗"（只能答是/否），多问"如果你是小兔，这时候会想什么"。
- **观点采择**："他为什么这样做？""他知不知道小伙伴会难过？"用扩展把回答接更丰富："你说他是因为害羞，对，他不敢说，所以躲起来了。"
- **联系生活（Distancing）**："你有没有过像小熊一样生气的时候？那时你怎么做？""下次和小朋友抢玩具，可以学书里的谁？"——这是开放式，不是"所以你以后要学小熊"的命令。

可参考的对话片段（以"玩具被抢、主角很生气"为例）：
- （情绪命名）"你看他的脸，现在是什么感觉？"——"生气。"
- （扩展粒度）"对，他气得脸都红了，这叫'非常生气'，还有点'委屈'，因为不是他愿意给的。"
- （观点采择）"他为什么后来还是把玩具递回去了？他心里怎么想的？"
- （联系生活）"你上次被拿走小车时，是不是也这么生气？那时你做了什么？下次可以试试书里哪个办法？"

若孩子不接话或答非所问：先接纳、不纠错，回到情节慢慢绕回情绪。对话质量永远比"问完所有问题"重要。

分龄建议：
- **3—4 岁**：重点情绪命名、识别基本情绪；多用填空和简单 Wh-；联系生活先连当下具体事件（"你今天有没有生气"）。
- **4—5 岁**：观点采择萌芽、冲突讨论、情绪粒度细化；多引导想"他为什么"。
- **5—6 岁**：自我调节策略、负责任的决定、"下次怎么办"；联系生活指向未来策略，讨论多种解法（"除了抢回来，还有别的办法吗"）。

选书标准：情节围绕明确情绪（生气、害怕、失落），包含角色冲突与解决，呈现不止一个视角。应覆盖情绪光谱，不回避愤怒、恐惧、嫉妒、分离这类"高唤醒"情绪——回避不会减少孩子体验，只剥夺了在安全语境下讨论的机会。

常见误区：只讲情节不讲情绪；用说教替代讨论（"你看，所以你以后不能这样"堵住孩子自己想的空间）；忽视孩子自己的真实反应（"这有什么好怕的"）；把绘本当教材逐页考核；以为"共读必然提升 SEL"（共读对语言/读写因果证据强，对 SEL 多为小样本、短周期研究，应视为有潜力、需刻意设计的实践，而非保证结果）。

## 六、建设性冲突解决（六步法）

家里两娃、班里抢玩具都能用。刚开始大人带着走，走过十几二十次很多孩子自己会用：

1. 走近，平心静气说清问题："你们俩都在抢这辆小车。"
2. 让每个孩子说感受和需求（句式"我想要……因为……"）："我想要先玩，因为我等了好久。"
3. 轮流想办法，尽量多提方案，不急着否定。
4. 一起评估："这样对大家都公平吗？安全吗？"
5. 共同选定一个方案试。
6. 事后复盘："这样解决了吗？下次怎么做更好？"

孩子不是被评判谁对谁错，而是在大人陪着走的过程中练出"我可以怎么和别人商量着解决"。

## 七、需关注的信号

以下情况建议寻求专业评估（注意：这是观察提示，不是给孩子下结论；专业评估才能区分"发展阶段的正常波动"与"需要进一步干预的情况"）：

- 某种信号频率高、强度大、持续时间长，明显影响吃饭、睡觉、社交、学习或日常生活；
- 上小学后突然不肯去学校、夜里尿床、总说肚子疼却查不出病、攻击或沉默突然变多；
- 持续的、全面的退缩，且较少主动表达需求（少数情况可能与语言发育、孤独症谱系特征等有关）；
- 成人自己判断"拿不准、很担心"时。

可依次考虑：园校心理老师或园医、社区儿童保健科、儿童医院发育行为儿科或儿童心理科、精神卫生机构儿童门诊。

## 八、与小学衔接要点

- SEL 是上小学的底层能力：能不能管住自己、和同学处、遇到困难会说，比"会不会算"更常被小学老师看重。
- 幼儿园阶段练的就是小学需要的"软能力"——被教会"生气可以、但要说、可以跺脚""怕分离可以带照片""想加入可以先在旁边看再慢慢来"，上小学面对新环境底气更厚。
- 衔接是从"随时被接住"逐步过渡到"有一定规矩"：大班开始慢慢加举手、等待、安坐一小会儿、自己整理书包；规则讲清为什么，而非"因为我说了算"。
- 别为了"看起来乖"压抑信号：健康标准是"遇到困难能表达、能找人帮"，不是"不惹事"。
- 小学把 SEL 嵌进晨会、班级公约、合作小组里，而非单开一门课；家校继续用"描述感受"代替"你就是不努力"。
