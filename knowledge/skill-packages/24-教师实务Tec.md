# 教师实务工具箱

面向幼儿园及小学低年级（3-9 岁）教师的日常实务：备课写教案、写评语、差异化与个别引导、不同特质儿童、观察评估、家园共育话术、融合支持、课堂突发应急。全部给模板与话术，照着用即可。

---

## 一、备课与教案设计

核心判断：教案是把"儿童发展"变成"每天能做的教育行为"。两个常见偏差要避开——小学化（把读写算直接搬进幼儿园）和空泛化（"培养兴趣"写不出、评不了）。

主题/单元教案模板：主题来源 → 核心经验（对照《3-6 岁儿童学习与发展指南》五大领域或一年级课标）→ 子话题与活动 → 环境与材料 → 家园共育 → 评估方式。

课时目标 ABCD 写法（对象+行为+条件+程度）：
- 例："中班幼儿（A）在提供 5 个不同大小瓶盖时（C），能独立按从大到小排列（B），正确率≥80%（D）。"
- 目标用可观察动词（布鲁姆层级：记忆/理解/应用/分析/评价/创造），别写"培养兴趣"这种空话。

区角观察记录表：谁 / 在哪个区 / 做了什么 / 持续多久 / 遇到什么困难 / 怎么解决的。

一日作息弹性表：固定大块（户外、区域、餐睡）＋ 可伸缩的"弹性时间"容纳生成活动。过渡环节用儿歌、音乐、游戏化指令减少等待。

生成课程（项目式）：从孩子真实疑问出发（如"影子为什么会动"），用 KWL 表（已知-想知-学知）梳理，按高瞻"计划-工作-回顾"组织每日探究；教师是"接住孩子抛来的球的人"，不硬拉回预设、也不被兴趣带着乱跑。

备课自查清单：
- [ ] 是否对照《指南》/课标写明领域或学科目标？
- [ ] 目标是否用可观测动词表述？
- [ ] 材料是否低结构、可操作、有错误控制（孩子自己能发现对错）？
- [ ] 是否预留生成空间与弹性时间？
- [ ] 幼儿园是否避免小学化、一年级是否零起点教学？

## 二、写评语

原则：评语不是打分，是让家长看见孩子的真实发展过程、让孩子从具体正向描述里建立"我是怎样的学习者"的自我认知。

发展性评语"看见—描述—期待"三段式：
- 看见：我注意到你最近在（具体情境）中……
- 描述：你（具体行为＋努力/策略），比如（一个真实细节）。
- 期待：接下来我们可以一起试试（贴合其水平的下一步）。

学习故事（注意—识别—回应）结构：注意（发生了什么、你做了什么）→ 识别（这展现了你的什么学习品质，如坚持/责任/合作）→ 回应（我们准备怎样支持你继续探索）。用"你"称呼孩子，聚焦"能做的"。

差异化标尺：
- 发展暂时落后的孩子：用鼓励型，落脚在"比之前进步了什么""已经能做到的小步"。
- 发展超常的孩子：用挑战型，提出"下一步可以探索的方向"，别只用"优秀"带过。

套话改写示例：
- 套话："小明这学期表现很好，很聪明，就是有点调皮。"
- 改写："我注意到建构区里，小明会用圆柱体积木搭出带旋转楼梯的城堡（看见）。你还教同伴'用拱形门更牢固'，愿意把办法分享给别人（描述）。下学期我们可以一起画一张'建筑物抗震设计图'，把你发现的稳固办法用到更大的项目里（期待）。"

评语自查：有没有具体行为而非空泛形容词？有没有聚焦"能做的"而非"缺的"？家长读完能否知道"下一步怎么支持"？

## 三、差异化与个别引导

分层任务卡（以"数数"为例）：基础卡（点数 1-10 实物并说出数字，给点数图）→ 标准卡（按指令取"比 5 多 3"的物件，给关键词卡）→ 挑战卡（编一道"分糖果"应用题，给提示句式）。层级可上可下，不贴"永远落后"的标签。

IEP 雏形（普通班也能用）：收集信息→写现状（优势+困难）→团队（教师、家长、专业人员）共拟长短期目标→落实策略→定期检核（建议每学年至少两次）。
目标写法 SMART：具体、可测、可达、相关、有时限。
- 反面："提高数学能力"。正面："12 周内，学生能正确书写 50 个高频词，正确率 80%"。
- 简易版："4 周内，能在小组活动主动举手表达 1 次/天，教师每日记录，每月复盘。"

气质与学习风格适配（Thomas & Chess 九维度；不是改造孩子，是让环境适配）：
- 视觉型：多图卡、示范、色彩标记。
- 动觉型：用身体动作学数数、摆教具、角色扮演；给"发教具员"这类职责把好动变贡献。
- 慢热型：给"预热"时间，新活动先个别告知、循序渐进。
- 易激/困难型：减少高刺激，给可预测流程和情绪调节支持。

误区：把"活泼"当"多动"（ADHD 需多场景、持续 6 个月以上且影响功能，由专业医生确诊）；IEP 目标不可测；固定分层伤自尊（用流动分层、异质协作）；只改内容不改环境。

观察记录，非测评结论；重大安排请咨询早教/幼教专业人士。

## 四、不同特质儿童引导

四类常态特质，都不是"问题孩子"，是发展阶段不同。教师提供适配环境，把行为往可预期、可共处、可合作的方向带。

攻击性行为（先止伤、再归因、后教替代，不贴"坏孩子"标签）：
1. 当下制止、分开双方、轻声安抚，保持中立不急于评判。
2. 冷静后用"观察-感受-需求-请求"四步复盘。
3. 教替代动作：想加入→"我可以一起玩吗"；生气→"我需要暂停/吹泡泡呼吸"。
4. 即时具体表扬替代行为："你刚才用嘴巴说'轮到我了'而不是抢，做得好。"
5. 用 ABC 记录表（前因-行为-后果）追踪功能，调整方案。
原理：用正向行为支持（PBS）教替代行为＋积极强化，比惩罚长期有效（惩罚短期看似有效，长期增加攻击性、破坏关系）。

退缩/内向儿童（慢就是快，scaffold 递进，不强迫）：
- Step1 一对一、有任务的互动 → Step2 一对一、开放式 → Step3 小群体（3-4 人）→ Step4 大型团体。
- 话术："你暂时还没准备好过去一起玩吗？我们可以先在这里看一会儿。""你刚才和小明一起完成了作品，这步跨得很好。"

多动/冲动儿童（先区分"活泼"和"ADHD"，再给结构化支持）：清晰作息、座位远离干扰源、任务分解、视觉时间表（图片化任务清单）、每日 20-30 分钟户外运动提升前额叶执行功能、正强化。若怀疑 ADHD，须转介儿童发育行为/精神科专业评估，教师不自行诊断、不贴标签。

规则意识弱儿童（不是"坏"，是规则未内化）：班级规则共建（班会讨论出 3-5 条正面表述并手写张贴）→ 视觉提示卡（洗手步骤、收玩具顺序、发言卡）→ 自然后果（推倒积木自己收拾，而非额外惩罚）。

筛查非诊断、疑点转介专业；不自行判定任何发展或行为障碍。

## 五、学业提升与观察评估

这个阶段真正要积累的是两件事：支撑终身学习的"学习品质"（专注、好奇、坚持、任务意识），和各领域的"学科核心经验"。过早小学化刷题，会牺牲好奇心、专注力和对学习的积极态度——《指南》明确指出这是"短视而有害的"。

早期数学核心经验（9 项）：集合分类 / 模式 / 计数 / 数符号 / 数运算 / 量的比较 / 测量 / 图形 / 空间方位。数感从一一对应、基数原则培养；用真实食物/教具在情境中理解运算，避免写算式。

早期读写核心经验：前阅读（良好阅读习惯、阅读理解策略、表达与评判三范畴）、前书写（从图画到符号到汉字，保护兴趣）、在阅读输入基础上自然渗透拼音识字，避免机械抄写。

学习品质培养：专注力融入"绘本寻宝""拼图接力"等游戏情境自然延长；任务意识用"班级小管家"轮流表；儿童版时间管理用计时器、一日流程图把时间"变看得见"，高年级可引入番茄钟雏形。

观察的几种方法（按目的选，不必贪多）：
- 叙事/轶事记录：客观写"谁、何时何地、发生了什么、说了什么、结果"，先记事实后写推测，同一行为至少观察 3 次再下结论。
- 时间取样：固定间隔记某行为"有/无"，看频率。
- 事件取样（ABC）：专记某行为的前因-行为-后果，看功能。
- 检核表：对照发展里程碑快速勾选。
- 作品分析：按 Lowenfeld 阶段（涂鸦期 2-4 / 前图式期 4-7 / 图式期 7-9），看表征方式、空间、顺序、情绪，而非"像不像"。

发展里程碑参考：CDC 里程碑（多数儿童某年龄能做到的事）、ASQ（家长填的筛查问卷，1 月-5.5 岁五能区）、《指南》五大领域。这些都只是筛查/参考，未达标提示"需关注"，不等于确诊；确诊由专业机构完成。

误区：小学化刷题；把活泼当多动；观察只记"好/坏"不记前因后果；里程碑当诊断；看到落后就贴标签。筛查非诊断、疑点转介专业。

## 六、家园共育进阶话术

核心：家园共育是成人与成人围绕同一个孩子的协作，不是"老师布置、家长执行"。政策依据包括《幼儿园工作规程》《纲要》《关于加强家庭教育工作的指导意见》《保育教育质量评估指南》均将家园平等互信合作列为要求；NAEYC 发展适宜性实践也将"与家庭建立互惠关系"列为核心原则。

非暴力沟通（NVC）四步：观察（只说可核实事实，不加"总是/太调皮"）→ 感受（表达真实情绪而非指责）→ 需要（说出背后诉求）→ 请求（具体可执行的行动，而非模糊要求）。

正向—具体—建议反馈结构：先锚定一个真实优点（不是客套）→ 用可观察行为代替形容词 → 给家长在家可操作的一小步。

场景话术对照（错误→推荐）：
- 入园适应："你家孩子哭得太厉害、适应能力太差"→"乐乐这周入园前 20 分钟会哭，但玩沙水区能安静下来。咱们保持一致：早上简短告别不回头，在家多聊幼儿园好玩的，我们每天发一段他开心的瞬间。"
- 行为问题："你家孩子太调皮，回去好好管管"→"这周集体活动有三次他离开座位动别人材料（观察）。我想试试用'小任务'让他当材料管理员（请求）。您在家和他练'想加入先说一句话'，我们同步观察？"
- 疑似发展滞后："他这明显是语言发育迟缓，赶紧去医院"→"我注意到浩浩说话词汇比同龄少些，多用拉手指物。园里多给他表达机会。方便的话带他去妇幼保健院做一次发育筛查，我把观察记录整理给您参考。"
- 安全事故："今天你家和别人打起来了，对方家长找来了"→"今天户外有件事要第一时间说：两个孩子抢滑梯推搡，明明额头红了，我们已冰敷观察，目前精神好无呕吐（事实+措施）。已分开引导重演'排队等一等'。想约时间细聊在家怎么练'轮流'。抱歉让您担心。"
- 与祖辈："爷爷奶奶别总惯着他"→"奶奶您喂得仔细是疼孩子。我们在练他自己用勺，已能舀大半碗。在家让他自己吃前几口您再帮，慢慢就全自己吃了，行吗？"

话术自检：说的是可观察行为还是贴了标签？有没有先接住家长情绪再讲事实？给的建议是否"小、具体、做得到"？是否把责任变成"我们一起"而非"你们家的问题"？涉及发展判断是否用了"观察/建议筛查"而非"诊断"？

## 七、特殊需要融合支持

政策依据：《残疾人教育条例》（2017）明确"积极推进融合教育、优先采取普通教育方式"；《"十四五"特殊教育发展提升行动计划》以"适宜融合"为目标；《关于加强残疾儿童少年随班就读工作的指导意见》要求"科学评估、应随尽随"；《普通学校特殊教育资源教室建设指南》要求普通学校（含幼儿园）建设资源教室。幼儿园属学前段，多以"个别化支持计划（ISP）/ 融合教育支持方案"形式落实。

ISP 简易流程：收集信息（观察+家长访谈+已有评估）→ 先写"能做什么"再写"需要什么"→ 定 3-5 条具体可观察目标 → 设计支持策略（环境/教学/行为/同伴）→ 分工执行（主班/配班/保育/资源教师/家长）→ 每 1-3 个月复盘。

常见障碍的幼儿园支持策略（不构成医学诊断，是否达诊断标准须专业机构评估）：
- 孤独症谱系：稳定可预测的日程＋视觉提示；清晰简短指令＋示范；从平行游戏到合作游戏配"同伴小老师"；尊重感官敏感设安静角；提前预告变化用社交故事。
- ADHD：座位靠近教师、减干扰；长任务拆短步配计时器；一致规则＋即时正向反馈；允许合理身体活动；视觉提示卡。
- 语言发育迟缓：扩展法（孩子说"车"，教师补"红色小车开走了"）；制造沟通机会（把想要的放看得见拿不到处）；配图沟通板；与语言康复目标对齐。
- 感统失调：设"自我调节角"（软垫、豆袋）；避免强制触碰；调整衣物餐具减敏感源；与专业感统训练衔接。
- 智力发育障碍：目标拆更小、重复更多、从具体到具体；多示范＋手把手辅助逐步撤除；强调功能性自理技能。

通用原则：调整的是"方式"，不是"降低标准"；先给"最少量的辅助"让儿童参与，再逐步撤除（脚手架式支持）。

从发现到转介流程：日常观察（连续 1-2 周，记具体事例）→ 园所内沟通判断是否为"跨情境、持续"困难 → 首次非正式家园沟通（先谈优势再谈关注点）→ 班级内先做基础调整观察 2-4 周 → 若持续且影响发展，建议家长带去残联定点机构/妇幼保健院/儿童医院做专业评估（是"建议"非"强制"）→ 拿到报告后更新 ISP、持续随访。

教师无权做医学诊断，只能做教育观察与转介建议。任何"你家孩子是不是孤独症/ADHD"都应改为"我们在某方面观察到一些需要多关注的表现，建议去专业机构评估确认"。

## 八、观察评估工具模板库

方法分级（按目的选）：A 级叙述性（叙事记录、学习故事）→ B 级取样（时间取样、事件取样 ABC）→ C 级量化（检核表、评定量表，最易凭印象打分，须以多次观察为依据）→ D 级作品分析。日常管理以叙事＋学习故事为主，阶段性建档用检核表，研究特定行为用事件取样，筛查预警参考 CDC 里程碑/ASQ。

幼儿行为观察记录表：幼儿姓名/代号 → 日期时间 → 观察情境 → 观察目的 → 客观记录（只写可验证事实，原话加引号）→ 我的解释（标注是推测）→ 下一步支持策略。填写守则：客观栏只写事实；同一行为至少观察 3 次再下结论。

发展检核清单（参照《指南》五大领域，示例性，实际以官方原文为准，勾选：已做到/发展中/暂未出现）：健康（大动作、生活习惯）、语言（倾听表达、阅读准备）、社会（人际交往、社会适应）、科学（探究、数学认知）、艺术（感受欣赏、表现创造）。此表不可当诊断标准；某领域多项"暂未出现"且明显偏离同龄，应结合 CDC/ASQ 进一步观察并建议转介。

学习故事写作模板：注意（今天在…我注意到你…）→ 识别（这展现了你的…因为…）→ 回应（接下来我们准备…帮助你继续…）。用"你"称呼，聚焦"能做的"，配真实照片，结尾给家长一句"发现"。

观察评估服务于支持与改进教与学，不是甄别或贴标签。筛查非诊断、疑点转介专业机构。

## 九、课堂突发与安全应急

首要目标不是"立刻压住行为"，而是先让孩子恢复安全感与冷静，再处理问题（低龄儿童前额叶未成熟，情绪激动时说理基本无效）。

冲突调解六步（抢玩具/打架）：先安全分开 → 描述事实"你们都想要这个" → 确认感受"很生气对不对" → 把问题抛回"只有一个，怎么让两人都开心" → 引导方案（轮流/一起玩/交换/沙漏计时）→ 执行并事后教"用话代替手"。不追"谁先动手"，不制造对立。

情绪崩溃：先共情、不急于讲道理；设冷静角（安静舒适的自愿角落，有软垫、沙漏、情绪卡、冷静瓶，不是惩罚、不是罚坐）；教情绪词汇（生气/委屈/害怕/失望）替代行为爆发；非言语提示（眼神、轻拍肩、手势、固定过渡音乐）比大喊少制造对抗。

离座乱跑：非言语提示＋固定过渡信号＋对小步进步即时具体表扬（"你刚才自己回到座位，很棒"）。

安全先兆排查：防撞角、地面防滑、窗户限位、围栏高度≥110cm、插座保护盖、圆头剪刀专人保管、午睡查口袋异物与蒙头、户外器材螺丝松动、过敏儿童单独备餐禁互分食物、师生比与交接班确认。把安全规则变成孩子能懂的图示与儿歌，日常渗透而非出事再教。

生理急症（呕吐/过敏/惊厥）：第一时间按园所预案处置并联系校医/120，教师不擅自做超出职责的医疗处置。过敏儿童入园建档案、单独备餐标姓名、禁互分食物；严重过敏黄金抢救约 5-10 分钟，园所应备肾上腺素笔并全员培训，发生时立即用、打 120、专人陪护。呕吐/惊厥让孩子侧卧防误吸、清周围硬物、记发作时长，不强行按压肢体、不往嘴里塞东西。

海姆立克与儿童 CPR 仅作科普：气道异物新近指南（2025 更新）建议优先 5 次背部拍击，再视情况腹部冲击；1 岁以下用"5 次背拍+5 次胸压"循环，严禁腹部冲击。实操必须经过正规课程培训并通过考核，未受训者先呼救、打急救电话、尽快取 AED。以下任何急救内容只作科普，实际操作务必受训并严格按园所/校医预案。
