# 16 · Base 概念怎么讲（edu-lesson 模块参考）

给家长/老师/教育者助手用的"概念讲法"笔记。核心目标：把抽象概念翻译成孩子能懂的说法，提供可直接套用的具体讲法示例。

---

## 一、通用套路（讲任何概念都用得上）

1. 从看得见的东西出发，不要先背名词或公式
   - 先拿孩子身边实物：冰块、锅盖、气球、镜子、自己的手。
   - 能触摸、能观察，概念才落得地；名词是入口，不是目标。

2. 用因果和功能解释，不贴标签
   - 不问"这是什么"，多问"它为什么在这里""它帮了什么忙""它怎么活下去"。
   - "地方和生活互相塑造""结构服务功能"——把知识放回关系里。

3. 先分大类/看共同点，再看差异
   - 动物先分哺乳/鸟/鱼/爬/两栖/无脊椎；细胞先看共同零件再看特殊设备；交通工具先按环境分。
   - 分类不是为了考试，是让孩子知道"世界各有来处"。

4. 用比喻和类比把抽象变可感
   - 原子分子：质子数像"身份证号"；分子像乐高，拼法不同东西就不同。
   - 人体系统：骨骼是积木架子、肌肉是马达、循环是快递、呼吸是倒着长的树、神经是电线、肾是过滤器。
   - 比喻只是帮理解的图，可顺带说明"不是真的"。

5. 身体参与、动作游戏胜过听讲
   - 反义词用站高蹲低、跑快走慢来演；感官用布袋摸物、纸杯电话来玩。
   - 抽象词变成可感受的经验。

6. 把抽象品质拆成具体动作
   - "有礼貌"=看着对方+清楚声音+等别人说完；"认真听"=停下手+看向说话人+听完再回应；"收玩具"=先找同类、积木进盒、车回停车场。

7. 允许多标准、重证据推理，不追求唯一答案
   - "哪个不属于这一组"追问"你按什么分"，理由成立就接受。
   - 猜动物从"是什么"改成"你怎么知道"，逼孩子用线索推理。

8. 留问题、保护好奇心，不急着给全答案
   - 先反问"你觉得呢"，再给一个能想象的类比画面。
   - 精确是后来的事，现在的任务是"哇"的表情和继续问的冲动。

9. 安全边界随概念一起讲
   - 交通、实验、玻璃、电路、海洋动物，都先讲"为什么这样才安全"，孩子理解原因才更愿意配合。

## 二、可直接套用的讲法示例

### A 科学概念

物质与三态
- 物质 = 有重量、占空间的东西。空气也是物质：吹鼓的气球说明它占地方；风能推车说明它不是"什么都没有"。
- 冰、水、水蒸气：像同一个人换三套衣服，核心还是水。融化/蒸发/凝结，分子没变，只是排列和运动变了。
- 物理变化 vs 化学变化：只换形状、状态（冰化水、纸撕碎）多半是物理；变成新东西（铁生锈、纸燃烧）是化学。判断法：有没有变成新物质，比"能不能变回去"更准。
- 纯物质 vs 混合物：水果沙拉能挑出来是混合；盐水蒸发盐能留下是混合；但 H2O 不是氢氧随便混，是化学结合，不能靠滤网分开。

原子、分子、元素
- 元素由质子数决定，不是由名字决定："质子数像身份证号，名字能翻译，质子数决定它到底是谁。"
- 分子是原子牵手：氢气可燃、氧气助燃，但按特定方式结合后形成能灭火的水——组合方式也重要，像乐高不同拼法变车变房。
- 不必背周期表，先养成三问：是不是物质？可能由什么小颗粒组成？是重新组合成新物质，还是只是混在一起？

力与运动
- 重力：地球从四面八方把我们拉住，"下面"对每个人都朝向地心。
- 惯性：东西有点"坚持原来的状态"，正在动就想继续动，停着就不想自己动。
- 摩擦力：球在地毯滚得短、在光滑地板滚得远，因为摩擦更大。可让孩子先预测再测量距离。

光和能量
- 能量是"让事情发生的能力"：手电筒亮是化学能→电能→光能；拍手是机械能→声能。
- 镜子=光被平滑表面反射；黑衣服更热=吸收更多光；放大镜变大=光折射。
- 影子：手电筒照玩具，靠近光影子大、靠近墙影子小，说明光沿直线传播。

地球科学
- 水循环："太阳把水请上天，冷空气把水变成云，云太重了就把水送回地面。"
- 白天黑夜=地球自转；四季=公转+地轴倾斜，用手电筒正照（光斑小亮）和斜照（光斑大淡）演示，不是太阳忽远忽近。
- 岩石三类：泥沙压出来（沉积岩，化石多）、火山冷却出来（火成岩）、被地球大力揉过（变质岩）。
- 板块像拼图分很多块，互相挤可能成山、分开岩浆涌出、一个钻另一个下面成火山地震。
- 风化=把饼干掰碎（原地碎开）；侵蚀=把碎屑搬走；沉积=把材料放下（堆出沙洲三角洲）。

太阳系与太空
- 太阳是中心不是背景：地球像小玻璃珠，太阳像大瑜伽球，里面能装下约130万个地球。
- 八大行星按"住哪里"记：内侧四颗岩石行星，外侧四颗巨大行星；每颗抓一个特点（水星最近最快、火星红色、木星最大有风暴、土星有环等）。
- 轨道：月亮绕地球、地球绕太阳，是速度和引力一起形成的运动结果；可用绳子系小球绕圈演示。
- 太空"没有重力"要谨慎：更准确是微重力/看起来漂浮，空间站一边被地球拉一边快速向前飞，所以绕地球转。
- 天体/地方/工具/现象先分四类，孩子才不会把望远镜和行星放同一层。

植物
- 光合作用："叶子接住阳光，根送来水，空气送来二氧化碳，植物在身体里做出糖。我们呼吸的氧气很多来自植物。"叶子是"厨房"。
- 根像吸管+仓库（胡萝卜、红薯是根的储存）；茎是运输通道（木质部送水、韧皮部送糖，像水管和送餐车）。
- 花不是为漂亮，是为制造种子；果实保护和帮助种子传播。
- 种子是"睡着的小植物"，得水、空气、合适温度就发芽；生命周期是一圈不是直线（果实不是结束，种子是下一次开始）。

动物与分类
- 分类从明显差异开始：哺乳（大多有毛、喝奶）、鸟（羽毛、下蛋、多数会飞）、爬（鳞片、产卵、体温随环境）、两栖（小时候水里长大上岸）、鱼（水里、鳃呼吸）、无脊椎（无脊柱，如章鱼螃蟹昆虫）。用"多数"而非"全部"，留例外（企鹅是鸟不会飞、鲸是哺乳动物不是鱼）。
- 线索推理看五类：外形、运动、食性、栖息地、特殊本领。企鹅：有羽毛是鸟、不会飞但会游泳、住寒冷海域、吃鱼和磷虾。
- 食性到生态位：草食/肉食/杂食；食物链"草→斑马→狮子"，但真实更像网。
- 本领接到适应：猎豹快为追猎物、章鱼喷墨为争取逃跑时间、海狸筑坝为创造安全环境。动物本领不是炫技，是帮它活下去。

恐龙与史前动物
- 史前动物是大盒子，恐龙是其中一个小盒子；翼龙会飞、沧龙鱼龙住水里，都不是恐龙。
- 身体线索推理：尖牙利爪多为肉食、 Flat 牙长脖子多为植食；三角龙三只角、剑龙背板尾刺、甲龙硬甲尾槌，都是防御工具不是装饰。
- 名字常藏特征：Triceratops=三只角、Stegosaurus=背板、Brachiosaurus=长脖子。
- 科学仍在更新：现在知道的恐龙是据化石推理，新发现可能改变旧认识，"不知道"不是失败。

海洋动物
- 不是会游泳就是鱼：鲨鱼是鱼（软骨、鳃呼吸），鲸和海豚是哺乳动物（肺呼吸、要上浮换气、哺乳），海龟是爬行动物（硬壳、上岸产卵），章鱼是软体动物，水母是刺胞动物。
- 身体形状回答环境：海豚流线型、海龟硬壳、章鱼柔软钻缝、比目鱼贴海底、河豚鼓起来。
- 食物链"浮游植物→小虾→小鱼→大鱼→鲨鱼"，真实是网。
- 安全：看见、观察、画下来可以；追赶、触摸、带回家不可以。

五感
- 视觉：晶状体把影像投到视网膜是倒着的，大脑翻正——眼睛里也有一块"变焦透镜"（用放大镜演示）。
- 听觉：声音是空气振动，经鼓膜、三块最小骨头传到耳蜗；太响会永久损伤听力。纸杯电话演示"线振动变声音"。
- 嗅觉：大部分"味道"其实是闻到的，舌头只辨甜咸酸苦鲜五种，复杂味道靠鼻子。
- 触觉：皮肤是最大感觉器官，指尖嘴唇最灵敏；布袋摸物猜是什么。
- 味觉：舌头没有"味觉地图"，任何味道可来自任何区域；孩子怕苦是进化保护（苦常关联有毒）。

人体系统（七类，都用比喻+触摸）
- 骨骼：积木架子撑起身体；头骨像头盔包大脑，肋骨像笼子护心肺；钙让骨头结实（牛奶里的钙会跑进去）。
- 肌肉：连着骨头，一收缩就拉动骨头；弯手臂摸到二头肌鼓起来"马达在工作"。
- 循环：心脏像拳头大、四个小房间，动脉送血出、静脉送血回；握拳松开模拟心跳"从在妈妈肚子里就没停过"。
- 呼吸：气管分叉像倒着长的树，肺泡像葡萄串小气囊；躺下把手放肚子，吸气鼓、呼气瘪，感受横膈膜。
- 消化：从嘴巴坐滑梯到胃，胃揉成糊糊，小肠吸走好东西，渣渣变嗯嗯出来，一趟约一天。
- 神经：大脑是总控制室，脊髓是主干道，神经像电线通全身；闭眼指身体部位，感受"大脑在导航"。
- 泌尿：肾像豆子过滤器，把血液里废物和多余水捞出来变尿，膀胱像气球胀了就想上厕所。

天气与四季
- 天气是"今天外面发生了什么"，每种天气对应身体感受和动作（晴天影子清戴帽、雨天穿雨鞋看水流、风天看旗子摆）。
- 风是空气移动：热空气轻往上跑，旁边冷空气补位就有了风。
- 四季是"一年里重复出现的变化"，做"季节观察册"每月画同一棵树，12张排一起看见一年在变。
- 每天出门前三问：今天天空什么样？身体冷热处理？需要带什么？把观察、感受、行动连起来。

材料与简单机械
- 电路：电像一队看不见的小跑者，必须有一圈完整跑道，路断了跑不过去；开关只是让路通或断。导体（金属）vs 绝缘体（橡胶塑料木）可做低压安全小实验；家庭插座不可碰。
- 磁铁：不是吸所有东西，特别喜欢铁镍钴；同极相斥异极相吸，中间那层"看不见的东西"是磁场；地球也像大磁铁（指南针）。电磁铁接电才有磁性，把磁和电连起来。
- 玻璃：主要来自沙子，高温下变软像很黏的糖浆（不是像冰一样有清楚熔点），工匠一直转是让软化玻璃受力均匀，吹气慢慢撑成空心，最后要慢慢冷却（退火）否则会裂。安全边界比作品重要，家庭只观察不操作。

### B 社会与空间概念

社区地点
- 地点不是建筑，是"一种社会功能"：问"这个地方在帮谁、帮什么忙"（学校帮孩子学、医院帮伤病、消防帮社区脱险、图书馆帮阅读）。
- 用"我们什么时候会去"建立真实连接（牙疼去诊所、借恐龙书去图书馆、轮胎漏去修车店）。
- 画儿童版社区地图（家、学校、公园、超市、医院），问"从家到公园经过哪""受伤了最近帮助在哪"。

职业
- 不问"你长大想当什么"，问"这个工作在帮谁、解决什么问题"。
- 按"帮助方式"分类讲：照顾身体（医生护士）、建造修理（工程师水管工）、交通（飞行员司机）、食物（农民厨师）、创作（音乐演员）、规则安全（警察消防员）、探索（科学家宇航员）。
- 讲三件事：在哪里工作、用什么工具、对谁负责（牙医=诊所+小镜子工具+护牙）。
- 打破性别刻板：职业看能力兴趣和训练，不看性别。

世界地标与地图
- 地标分四类（自然景观/历史建筑/现代建筑/信仰文化空间），重点讲"为什么在那里"：金字塔连尼罗河文明、雨林连热带气候、风车连低地排水。
- 地图是"从上往下看的约定"：先把床桌椅画成顶视图，再去看城市国家。
- 地名和方向、距离、位置连起来学；目标不是听过名字而是"我能找到它"（找家、找路线、找相邻）。

交通工具
- 先按移动环境分（路上/工地/水上/空中/水下），环境决定设计。
- 每种都有任务：救护车快速送病人、消防车带水梯救援、垃圾车保持社区清洁；"如果任务换了，车要变成什么样"是工程思维雏形（结构服务功能）。
- 安全随认知一起讲：听到警笛让路、头盔护头、等车站在线后、工程车盲区多不靠近、救生衣不是装饰。

体育
- 先讲"场地决定怎么玩"：球往哪去、人站哪、哪算出界、从哪开始、到哪算完成——场地是规则一部分，培养空间意识。
- 每人不同任务都重要（投球/击球/接球/跑垒/防守），团队不只有得分的人。
- 把动作拆成可观察的身体控制（眼睛看目标、身体转向、手臂前后），背后是手眼协调和平衡。
- 轮流是关键规则能力：你投一次我投一次、轮到别人我站安全位置看。
- 安全边界比胜负先讲（软球泡棉棒替代硬球）；输赢可讲但不放第一位，多问"你有没有看着球、等到轮到、按顺序跑"。

反义词
- 不是背词对，是理解关系：说"大"因为有"小"比较。
- 用身体演（站高蹲低、跑快走慢、装满倒空），抽象词变可感受经验。
- 有些是程度不是绝对：苹果比葡萄大比西瓜小，引向连续量理解。
- 也教感受反义词（紧和松、害怕和勇敢），帮孩子描述内在状态。

### C 品质与习惯培养

好习惯
- 先分四类：照顾身体（洗手刷牙）、照顾关系（请谢谢分享）、照顾环境（收玩具）、照顾秩序（认真听排队）。
- 每个习惯有触发场景：洗手=饭前/外出回/上厕所后；说please=想请人帮忙时。触发越固定越自动化。
- 抽象词翻动作（见上方套路第6条）。
- 用固定流程代替催促：饭前（停玩具→洗手→擦干→坐桌）、睡前（收玩具→洗→刷牙→睡衣→书→关灯）。
- 反馈要具体描述动作（"你饭前自己去洗手了"），不说空泛"真棒"；只说乖孩子学到讨好，说清动作学到行为与结果关系。
- 一次只抓1-2个高频习惯，稳定再加；允许孩子有边界（"这个我还想玩，计时器响给你"）。

礼貌
- 每句话背后有一个关系动作：请=我在请求不是命令、谢谢=我看见你为我做事、对不起=我知道影响了你、可以吗=我尊重边界。
- 大人先示范说出来给孩子听，比要求更有力。
- 道歉含修复三步：看见影响（我撞倒你积木）→说对不起→做修复（帮你重搭）。
- 等待不打断是尊重别人时间，给具体支架（"话放手里等他说完""碰下手表示有话"）。
- 礼貌也包括对环境和动物的温柔（垃圾入桶、图书馆小声、轻摸动物、椅子推进去）。

朋友相处
- 朋友可以一样也可以不一样，一样容易靠近，不一样学到新东西。
- 分享可操作的是轮流，给句子：想玩的人"我可以玩一下吗/你玩完给我吗/我等你两分钟"；正在玩的人"我还想玩会儿/你可以先玩这个/一起玩吗"。
- 被拒绝有下一步：问何时玩完、找另一玩具、请大人协调、等会儿再问——保护两个孩子（一个能说不，一个不崩溃）。
- 修复关系三步：看见影响→承担动作→修复语言，社交学习不在情绪失控时做。
- 帮助是具体动作：捡起朋友掉的东西、递一块拼图、问"需要帮忙吗"、说"你做到了"。

情绪
- 命名驯服：能说出"我很生气"而不是直接打人，是巨大飞跃——词语把情绪从身体搬出来变成可谈论的东西。
- 教学时机不是他崩溃时（前额叶关闭），而是睡前、绘本角色有情绪时、自己有情绪示范时。
- 工具：情绪脸谱图（问"你今天哪个脸"）、情绪温度计（1平静到5爆炸，"变红前可以做什么"）、读绘本暂停问"他现在脸在说什么"。

日常流程
- 可预期顺序替孩子减轻选择负担，少慌少被催。
- 画流程图（起床→刷牙→穿衣→早餐→书包→出门），每完成一步移动小夹子，视觉提示比催更友好。
- 早晚各抓5步就够，太长会迷路；过渡提前提示（"还有5分钟收玩具""计时器响去洗澡"）是帮大脑准备换挡。
- 稳定不是僵硬，变化时要讲清"今天流程变一点，因为……回来还是洗澡故事睡觉"。

分类思维（通用能力）
- "哪个不属于这一组"重点不是答对，是学会提标准：我按什么分、为什么它不属于。
- 允许多答案（西兰花按蔬菜不同、按要不要剥皮香蕉不同），前提理由成立。
- 标准从具体到抽象：外观→用途→场景→类别→属性；孩子只按颜色分不急，多问"它们哪里像哪里不一样"。
- 语言训练：把"这个不一样"补成"我选X，因为其他是A它是B"，练因为/其他/只有/属于等词。

## 三、方法底线（每次讲概念都守住）

- 保护好奇心优先于给答案：先问"你觉得呢"，再给能想象的类比，留问题比给全答案更有价值。
- 科学方法六步（孩子不必记词，但可学顺序）：提一个能验证的问题（可观察/可比较/可重复）→收集信息→做"如果…那么…因为"的假设→公平实验一次只改一个条件→记录分析"我看见了什么"→报告分享四句话（问题/假设/怎么做/发现）。猜错不是失败，是假设要修正。
- 安全随概念一起讲：实验先说边界（先读步骤、不懂就问、护目镜、不吃喝材料、电和热有大人在、意外马上说）；交通、材料、海洋动物同理。
- 不把影视/动画形象当标准答案，提醒"电影负责好看，科学要看证据"。
