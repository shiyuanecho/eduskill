# 教师实操工具箱

面向幼儿园教师的日常可用工具：一日生活怎么渗透学习、低结构材料怎么玩、观察记录怎么写、小组与个别化怎么指导、提问与对话怎么接、数学区角怎么设、项目与生成课程怎么落地。全部给模板与话术，照着用即可。

---

## 一、一日生活中的学习渗透

核心判断：幼儿从入园到离园的每分每秒都在学习，不是只有集体教学那二三十分钟。把生活环节只当"流程"会浪费大量练习自理、社交、语言、数学的机会，尤其对发展暂时落后的孩子，生活环节是最自然低压力的个别化训练场。

环节—可渗透发展点（通用地图，按本班实情改）：

- 入园问候：社会性情绪、语言。练主动打招呼、辨认情绪、平和道别。
- 晨间/早操：大肌肉、协调性、规则意识。
- 点心：数学（数数、分类、一一对应）、自理、表达需求（"我要一块"）。
- 盥洗如厕：顺序与自理、身体控制、独立性。
- 自由游戏/区域：所有领域，尤其象征游戏、问题解决、同伴协商。
- 户外：大肌肉、平衡与冒险、空间、自然观察。
- 午餐：自理、数学（分碗筷）、餐桌礼仪、食育。
- 午睡：自我调节、身体休息、依恋安全。
- 过渡环节：情绪平稳转换、听信号、用儿歌稳定秩序。
- 离园：整理收纳、回顾一天、社会告别、责任。

落地七步：

1. 画自己的"一日生活地图"：把孩子一天列出来，每环节写"他在这可能练什么"。
2. 先挑两三个环节做"显性化"，不一次全改。孩子数数弱，就重点在点心和分碗筷让他自己数。
3. 步骤用"说—示—退"：边做边说步骤 → 做给孩子看/手把手带 → 会了就只卡住时提示一句 → 最后完全让他自己来。
4. 用稳定信号管过渡：固定一首收拾歌/一个摇铃/一个口令，每天一样；情绪易崩的孩子提前三分钟预告。
5. 在生活里嵌数学和语言但不考：分点心"一人一块"练一一对应，摆碗筷"几人几双筷"练数数，收玩具"大积木放这筐、小车放那筐"练分类；让孩子自己说"我要/我还要/我够了"，离园请他说"今天最喜欢玩的是……"。
6. 给"冒险"留空间：允许尝试能力边缘的活动（一级台阶跳下、走平衡木、爬低矮攀爬架），成人站可及处保护但不抢先把危险抹掉；安全底线明确（不推挤、不从高过身高的地方跳）。
7. 离园前用回顾串一天："今天吃了什么、玩了什么、认识了谁"，帮孩子整合记忆，也让家长接到具体分享。

常见误区：只有上课才学 / 生活环节能省则省（包办反而亏了后劲）/ 过渡只要安静排队（核心是平稳转换不是压制）/ 生活环节不需设计（材料要够得到、步骤图示上墙、信号一致）/ 在生活里加学习就是硬塞知识（应"顺势显性化"，不追加考题）。

## 二、低结构材料与绘本资源（不列具体书名/品牌）

低结构材料：没有标准玩法、怎么折腾都行的东西（纸箱、绳子、布、石头、松果、泥沙）。对照"高结构"成品玩具（玩法被厂家定死）。开放可变、能组合的材料更能养创造力、想象力和问题解决；同一筐材料不同年龄孩子各取所需，是集体里难得的"一个环境多种发展"。

投放原则：成人经过思考投放——有粗有细、有软有硬、有自然物有人造物、数量充足能组合；分筐收纳贴图标识让孩子自己取放；定期轮换；成人角色是补材料、观察玩法、在需要时抛一个问题，不是规定玩法、当导演。

材料类别（在家收集或低价采购即可）：

- 自然物类：平滑石头/鹅卵石（排序、数数、摆图案、当棋子）、松果/橡果/干净无刺树枝（搭建、分类、拼贴）、落叶花瓣（拓印、颜色分类）、沙水泥（感官与空间探索）。三岁以下防误吞，泥沙防生食。
- 纸类：各种大小纸箱（钻、搭、画、剪窗）、卷纸芯纸筒（管道、望远镜、小动物）、报纸牛皮纸（揉团、撕贴、包裹）。纸箱拆边角防划手、大箱留通风口。
- 布与绳类：碎布旧手帕丝巾（披风、包扎、拼贴、触觉）、粗绳毛线（摆形、测量、编织初体验）。选可水洗、无细长绳带防勒缠，长绳成人看护防缠绕。
- 瓶罐与扣件类：干净瓶盖/小盒（分类、堆叠、角色道具）、木夹衣夹（夹物、数数、精细动作）。三岁以下禁用小直径瓶盖防吞咽。
- 建构类（可购）：单元积木/木质积木（搭建、对称、测量，建议作基础投入）、大颗粒连接类（自由造型，选大颗粒防误吞）。
- 角色与表征类：废旧电话/无电池计算器（假装游戏）、素色简易款小人偶动物模型（讲故事、分类，避免过多声光干扰）。

绘本使用方向：

按年龄：0–2岁选布书/洗澡书/硬纸板书，画面大色彩单纯，重点是一起翻、指认、说名称；2–4岁选情节简单重复句式、贴近日常的，可开始对话式共读；4–6岁选有情节、能讨论"为什么/如果是你会怎样"的，以及科普、情绪、友谊主题，可延伸手工与低结构玩法。

按主题：情绪与社会（命名情绪、练换位思考）、自然与科学（读后去户外收集对应自然物做拼贴）、数与形（读后玩瓶盖分类、石头排序）、语言与想象（句式重复留白多，鼓励接话编结尾）、生活习惯（温和建立常规）。

使用方法（对话式而非认字式）：读前先看封面猜"这是谁的故事"；读中停一下问"你在这看到了什么""你猜接下来呢"；读后用低结构材料做延伸（纸箱搭故事里的房子、石头摆角色）；反复读同一本没关系，每次聊的点可不同；让孩子翻页、指图、替角色说话。

常见误区：玩具越贵越益智（被动高结构电子玩具常挤占主动想象时间）/ 材料要成品化漂亮化（太像展品会压制探索）/ 绘本就是认字书 / 绘本越多越好 / 低结构材料脏乱不安全（合理筛选+成人看护即可）/ 随便给一堆破烂（要有组织地投放）/ 只重教不重玩（孩子主导）。

## 三、学习故事观察记录（模板）

核心思路：不给孩子打分排名，而是把某个真实时刻里"正在发生的学习"用故事写下来，看清背后体现了什么能力，再想下一步怎么支持。价值在于改变看孩子的眼光——先看见能力，再谈发展点。

三段式结构（注意—识别—回应）：

- 注意（我看到了什么）：只描述不带评价。时间地点、孩子具体做了什么说了什么、持续多久重复几次、哪个动作哪句话最关键。像用文字拍短视频，不下结论（写"他蹲了八分钟，积木倒了两次又继续搭"，不写"他很专注"）。
- 识别（这当中体现了什么学习）：把具体行为连到"学习"上。常用五类学习倾向：产生兴趣 / 投入其中 / 面对困难挑战不确定时坚持下去 / 与他人交流 / 承担责任。一句话解释为什么算学习、孩子在建构什么理解。贴泛标签等于没识别。
- 回应（下一步怎么支持）：基于前面看见与识别，想接下来怎么做。环境材料上调整什么、互动上问什么说什么、要不要请孩子回看故事说当时想法、后续观察什么变化。回应可来自老师、家长或请孩子自己。

【可直接照填的模板】

标题：给故事起个有画面感的名字

一、注意（我看到了什么）
- 时间/地点：____年__月__日，____区/____活动
- 当时情景：白描写孩子做了什么、说了什么、周围怎样（不评价）
- 持续时间/重复次数：做了多久、重复几次
- 关键细节：哪个动作、哪句话最让你觉得"有东西"

二、识别（这当中体现了什么学习）
- 连接到学习倾向（五类之一或几条）：____
- 一句话解释：为什么这些行为算学习，孩子在建构什么理解

三、回应（下一步怎么支持）
- 环境/材料上：可调整或新增什么
- 互动上：老师/家长可以问什么、说什么
- 请孩子参与：要不要让他回看这个故事、说说当时想法
- 后续观察点：下次重点看什么变化

写作要点：先别追求每天写，一周认真写一两篇比流水账强；把"注意"当摄像，只写动作和原话不写形容词；写完读给孩子听，若他说"对，我当时就是想试试"说明识别对了。不怕"倒、塌、错"，怕的是"看不见"——卡住和混乱里藏着最浓的学习。

常见误区：观察就是挑错（要找学习发生处，不是罪状簿）/ 文辞华丽最重要（第一段必须是事实）/ 只记成功可爱的（卡住时刻最有价值）/ 识别随便贴标签 / 当成给家长的表扬信只写优点回避发展建议。

## 四、小组与个别化指导

小组活动：3–6人围绕共同任务一起做，有共享目标、有分工协商（"一起搭一座大房子"）。不等于"大家各做各的坐同一桌"（那是分组不是小组）。核心是同伴互学——差异正是合作学习的发动机，强的通过解释巩固、弱的在近侧支持里往前走一步。

个别化指导：针对某个孩子当下发展水平给"刚好够得着"的一小步。不是一对一补差、不是把弱的单独拎走补课，而是日常渗透式支持（集体活动时蹲下说一句"试试先从大的开始拼"；点心时提醒总忘说谢谢的孩子）。它描述的是"支持的精度"，不是人数。

支架四招（连接两者的桥梁，节奏是"给到刚好够然后退"）：

- 示范：你先做一遍边做边说，孩子靠模仿起步。
- 提示：不直接给答案，给线索（"门要在哪儿呀？想想人从哪进"）。
- 追问：顺着他的话往下问推深（"你说小熊饿了，那它饿了会怎样"）。
- 撤支持：会了就逐步少帮，从手把手到动嘴到沉默。撤是目的。

落地七步：

1. 先判断适合小组还是个别：需不同想法碰撞/一起完成→偏小组；是某孩子具体卡点需精准推一步→偏个别。很多活动两者都要（先小组，过程中个别化巡视）。
2. 小组建法：3–6人舒服规模；任务"共同可达成"但留空白让孩子定细节；给不同角色降冲突（"你负责找长积木，你负责把门留出来"）；开场一两句说清就退到观察者，只在真卡住或起冲突时用提示介入；结束必有"一起看成果"环节。
3. 个别化落地不贴标签：一看（日常玩和做里看卡在哪）二接（当前水平加一小步）三记（记"今天数到6了"以便下次推哪步）。全程在群体里顺手做。
4. 把支架四招用熟：示范→提示→追问→撤。给多了孩子懒，给少了挫败。
5. 用"最近发展区"当尺度：独立能做（已会不帮）/ 踮脚够不着（太难先降档）/ 差一点点能行（正好在区里给支架）。只在第三种用支架。
6. 观察—调整闭环：谁没参与、谁被推过头，记下来下次调。个别化不是一次定终身，跟着每天小变化微调。
7. 别让小组变"强的带弱的单向劳动"：给弱的孩子也可贡献处（力气大的负责搬大积木），合作双向受益。

常见误区：个别化=一对一补课（贴标签伤自尊）/ 小组=分开坐（无互动则无价值）/ 能力不同的不能同组（差异是资源）/ 支架=多帮帮到底（灵魂是临时可撤）/ 迷信某一种（按目标和孩子状态灵活切换）。

## 五、师幼提问与对话技术库（可直接用的句式）

核心：会提问比会讲更能引发思考。开放式问题（无唯一标准答案，怎么答都算数）让孩子从答题者变讲故事和想办法的人；封闭式"是不是/对不对"孩子只需点头，脑子几乎不动。

关键动作——等待时间：问完值得想的问题后默数三秒（两三岁延长到五秒），忍住不抢答。孩子"那个……那个……"卡壳时正是大脑费力工作，一插嘴就中断。可双手交叠身前提醒自己也在等。

五类问题（配可直接用的句式）：

- 预测类（练推测）："你猜接下来会发生什么？""敲门之后呢？"
- 比较类（练找异同）："你发现这两块有什么不一样？"
- 解释类（练说清为什么）："为什么从这头推下去跑得更远？"
- 假设类（练反事实思考）："如果我们想让它更不容易倒，可以怎么做？"
- 评价类（练判断带理由）："这两幅画你更喜欢哪幅，为什么？"

九招实操：

1. 数三秒：问完默数一二三，两三岁数五秒。
2. 用"你说呢/你猜呢"代替"是不是"：孩子指鸟，别回"对是鸟"，试"你听它叫声什么样""它飞得高不高，和刚才那只要比呢"。拼错别急说"放错了"，试"你看这一块边边是直的还是弯的，应该去哪"。
3. 追问"为什么"和"还有呢"：像好奇的小伙伴不是审讯官。
4. 接住—扩展—归还（黄金三步）：接住（"你发现蚂蚁排着队"）→扩展（"它们搬的好像是面包屑，要搬去哪呢"）→归还（"你觉得呢"）。
5. 把错误当线索不当事故：孩子猜错先说"这个想法有意思"，再把问题抛回去验证（"你说天黑是太阳回家了，那明天一早看看太阳在不在天上"）。
6. 在一日生活埋对话钩子：入园问"你今天来路上看到什么好玩的"；点心问"今天的苹果和昨天的比哪里不一样"；散步问"你听现在有什么声音""那片叶子怎么掉的，你猜"；冲突时问"你们俩都想玩，有什么办法两人都能玩到"；离园前问"今天哪件事你最想讲给妈妈听"。
7. 给自己做对话笔记：每天挑一个环节记一句"我问了什么好问题、孩子回了什么、我是不是又抢答了"。
8. 分年龄微调：两三岁更具体可指认（"你指指哪个是你最喜欢的"）；四五岁上假设和评价；五六岁引入反事实（"要是故事里的人做了不一样选择，结局会怎样"）。
9. 同伴对话也重要：孩子之间聊起来，先旁边听，卡住或要发散时再轻轻抛一个问题。

常见误区：多提问=多考问（封闭问题机关枪）/ 问完立刻给答案或替说完 / 只肯定对的、悄悄惩罚错的 / 把对话变单向灌输包装（"是不是要排队呀""是"）/ 以为会聊天是天赋学不会 / 只在大组才提问忽略一对一和过渡环节 / 对孩子"胡说八道"失去耐心（先惊喜"想法好特别"再轻轻拉回可验证层面）。

## 六、数学区角活动设计

数学区角：专门腾出的小角落，放手能摆弄的材料，让孩子在玩中接触数学。不是上课刷题，是把数学藏进每天会做的游戏里。比刷题更能长"数感"——对数量的直觉（少了一个凭直觉知道，不用逐个数）。要区分"唱数"（背数字如唱歌，不知对应几样东西）和"点数"（手指点一个说一个数，最后说总数，连起数字与数量）。

区角目标三层：基础层（愿意靠近数学不害怕）、能力层（数概念、空间感、逻辑模式）、习惯层（"我会试、我会想办法"的探究劲，最易被忽略却最影响长远）。

低成本材料：积木一筐 / 带孔串珠加粗线 / 厨房量杯两个+小漏斗+装水盆 / 手写数字卡1–10 / 七巧板 / 六面骰两三个 / 玩具果蔬+自制纸币+小收银本 / 分类小筐 / 记录纸和蜡笔。

五个可直接用的活动（每项含"怎么玩—观察什么—怎么往前推"）：

活动一 小超市买卖（认数、对应、比多少）：摆玩具果蔬贴手写价签，孩子当老板数清几根、找对应价签收钱；熟练后给大钞练找零。观察：是否会数实物、能否把"三根"和"3"对上（数量与符号对应最难也最关键）、有没有比多少意识。往前推：只会递东西不看价签，就一起指认"这根写3，你收几块"；能收钱就故意给大钞陪他数"收5块东西3块，还我几块，一块块数回去"——陪他数不清替他报答案。

活动二 图形拼贴与七巧板（形状、空间）：低龄用裁好的三角形正方形长方形贴房子（正方形身体、三角形屋顶）；大一点照图拼或自由拼。观察：认不认形状、知不知道两个小三角形拼成大正方形、会不会旋转翻转组合。往前推：只会平铺不会组合，拿两块三角形示范"斜边对斜边变大方块"做一遍就收手；会照图就鼓励不照图拼自己的，把模仿推向创造。

活动三 量杯倒水比多少（容量、比较）：两形状不同杯子（细高/矮胖）倒水，问"哪个水多"。多数孩子脱口"细的高所以多"，不急着揭穿，让他倒回大量杯比刻度或倒进同杯看水位，自己发现"一样多"（守恒萌芽）。观察：是被外表骗住还是愿用倒来比验证。往前推：认定高的多就一起倒进大量杯"量量看"；能比出一样多就加第三杯排"从少到多"。

活动四 骰子走格子（点数、规则）：纸上画十格路，轮流掷骰子走几格先到终点赢，可加"踩红格退两步"。观察：能否看见骰子点点就说几（点数识别）、懂不懂轮流按点数走守规则。往前推：小班掷出4却乱走，握他的手一起点数"1、2、3、4，走4格"数完松手；稳定后引入两骰子加起来走（数两堆点子合起来数总数，不是背加法表）。

活动五 按规律串珠（模式、逻辑）：大人示范红蓝红蓝穿，问"接下来什么颜色"；换"两红一黄"；再让孩子自己设计规律给大人猜。观察：能否发现重复单元、说出"规律是红蓝红蓝"（用抽象方式描述秩序，接小学找规律甚至代数思维）。往前推：只会跟上一个颜色说不出规律，指给他看"红蓝红蓝一直重复，下一个应该是——"；能稳接就让他当规律设计师，你故意穿错看他会不会发现纠正，把被动跟变主动监控。

贯穿两条：向前推一点永远在孩子"差一点点够得着"处（最近发展区）；大人出手尽量"示范一遍+提问+退后"而非直接给答案。大班下学期可有意识把实物操作往画图记录过渡（买卖后画"今天卖3根香蕉收3块"），平缓跨进小学的具象到抽象台阶。

常见误区：区角只是摆设随便玩（自主≠放任，材料组合和介入有讲究）/ 只放数字闪卡（数字须和真实东西待一起）/ 完全不介入就是尊重（看准了再出手，跳过比多少补差额就浪费了）/ 材料越贵越好（低成本生活材料反而没距离感）。

## 七、项目式与生成课程落地

核心：幼儿学习大多发生在"他自己想知道"时。生成课程是课程观——课程从孩子那长出来；项目式学习是做法——用有头有尾的深入调查把兴趣落到实处。两者常是一回事：用生成眼光发现"孩子都在看蚂蚁"，用三阶段把"蚂蚁项目"做成持续探究。

项目三阶段（Katz & Chard 框架）：

- 启动（beginning）：唤起兴趣、收集孩子已有经验和疑问。蹲下问"你们想知道什么"，用便签记下问题，不马上回答。
- 展开（fieldwork/development）：实地调查、找资料、请教"专家"、用画/搭/写/演多元表征自己的发现。这是项目身体。
- 总结（concluding）：整理成果做展示/出小书/开分享会，回头看"我们学到了什么"，把未完成也当成诚实的科学态度。

【完整示例：蚂蚁项目（混龄四到五岁，约两周）】

启动：孩子看蚂蚁搬家争论，老师蹲下拿便签"把想知道的记下来"，记九个问题（住哪/怎么认路/吃什么…），说"我们当一周小蚂蚁研究员"。当晚给家长发照片和问题纸，请带相关绘本或材料。

展开：第二天科学区放放大镜、透明饲养盒、记录本，抛"你看到它用哪里说话"引导注意触角；第三天借蚂蚁绘本读"用气味认路"，做柠檬皮盖气味看蚂蚁绕开的实验；第四天请门卫叔叔讲蚁巢，孩子画"蚂蚁城"；第五到七天美工区合作搭泥制蚁巢模型分"宝宝房/食物房/通道"，粘土捏蚂蚁数清"头胸腹三节六腿"对照书确认，语言区口述"蚂蚁日记"。每天用学习故事记一两篇。

总结：把问题纸、观察画、模型、日记、实验照片汇成展板"我们当了一周蚂蚁研究员"，每个孩子选作品讲给别班听；老师问"九个问题现在能答几个"，没答上的坦白"还没弄清楚，下次接着研究"。结束发家长图文记录，附"孩子练了观察、提问、合作、表达"。

实操经验：

- 项目前囤"万能材料"（放大镜、各种纸、泥、胶带、绘本区留空位），兴趣来了接得住。
- 问题纸一定留着，是项目指南针，也是总结时最有说服力的证据。
- 别怕跑偏：孩子从蚂蚁聊到"下雨蚂蚁知道吗"可能正好生小分支，顺着走比硬拉回主线更有收获。
- 每次结束写简短复盘：哪步孩子最投入、哪步我介入太多、下次改什么。

老师在每个节点只做三件事：提供材料、抛好问题、帮着总结。

常见误区：生成=老师啥都不用准备（其实要求更高，要随时有材料、懂一点孩子可能问的、会判断兴趣值不值得做项目，否则变放羊）/ 启动后就指望孩子自己深度学习（老师要抛问题帮总结，否则热闹一阵就散）/ 做成手工展示秀（表征应服务"我探究后的发现"，讲不出"我发现/学到"就还停在表面）/ 只挑好看安全好展示的题（绕过孩子真起劲但有点脏有点难的问题，等于丢最有探究价值的时刻）/ 越长越复杂越好（长度由孩子投入度定，不是计划）。
